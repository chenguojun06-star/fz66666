# Grafana 监控使用说明

## 当前状态
✅ Grafana 已成功启动并运行
✅ 访问地址: http://localhost:3000

## 如何查看监控仪表盘

### 步骤1: 进入仪表盘页面
在左侧菜单栏中，点击 **"Dashboards"** (仪表盘图标)

### 步骤2: 浏览仪表盘
点击 **"Browse"** (浏览)

### 步骤3: 选择仪表盘
您会看到 **"服装供应链系统监控仪表盘"** ，点击它

---

## 仪表盘显示内容

### 顶部统计卡片 (4个数字)
| 指标 | 说明 |
|------|------|
| **CPU 使用率** | 服务器CPU占用百分比 |
| **JVM 内存使用率** | Java程序内存使用百分比 |
| **活动线程数** | 程序运行的线程数量 |
| **HTTP 请求速率** | 每秒处理的请求数 |

### 中间图表 (2个曲线图)
| 图表 | 说明 |
|------|------|
| **JVM 内存使用趋势** | 内存使用量的变化曲线 |
| **HTTP 请求响应时间** | 接口响应速度的变化 |

### 底部统计 (3个面板)
| 面板 | 说明 |
|------|------|
| **数据库活跃连接数** | 正在使用的数据库连接 |
| **数据库空闲连接数** | 空闲的数据库连接 |
| **HTTP 状态码分布** | 请求成功/失败的比例 |

---

## 颜色说明
- 🟢 **绿色** - 正常状态
- 🟡 **黄色** - 警告状态 (需要注意)
- 🔴 **红色** - 危险状态 (需要处理)

---

## 自动刷新
仪表盘每 **5秒** 自动刷新一次，显示最新数据

## 时间范围
默认显示最近 **1小时** 的数据，可以点击右上角调整时间范围
