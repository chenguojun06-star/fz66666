## 🔍 已知问题检查

### 检查1：扫码防重复逻辑

```bash
# 查看最近的扫码记录间隔
docker exec fashion-mysql-simple mysql -uroot -pchangeme fashion_supplychain \
  --default-character-set=utf8mb4 \
  -e "
SELECT 
  order_no,
  process_name,
  scan_time,
  TIMESTAMPDIFF(SECOND, LAG(scan_time) OVER (PARTITION BY order_no ORDER BY scan_time), scan_time) as interval_seconds
FROM t_scan_record
WHERE order_no = 'PO20260122001'
ORDER BY scan_time DESC
LIMIT 10;
"
```

### 检查2：订单状态流转

```bash
# 验证订单状态是否正确更新
docker exec fashion-mysql-simple mysql -uroot -pchangeme fashion_supplychain \
  --default-character-set=utf8mb4 \
  -e "
SELECT 
  order_no,
  status,
  progress,
  completed_quantity,
  total_quantity,
  (completed_quantity / total_quantity * 100) as completion_rate
FROM t_production_order
ORDER BY create_time DESC
LIMIT 5;
"
```

### 检查3：数据一致性

```bash
# 对比订单总数和扫码记录总数
docker exec fashion-mysql-simple mysql -uroot -pchangeme fashion_supplychain \
  --default-character-set=utf8mb4 \
  -e "
SELECT 
  o.order_no,
  o.total_quantity as order_total,
  COALESCE(SUM(s.quantity), 0) as scanned_total,
  (COALESCE(SUM(s.quantity), 0) - o.total_quantity) as difference
FROM t_production_order o
LEFT JOIN t_scan_record s ON o.order_no = s.order_no
GROUP BY o.order_no, o.total_quantity
HAVING difference != 0;
"
```
