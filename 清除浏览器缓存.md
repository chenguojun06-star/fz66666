# 🔧 清除浏览器缓存指南

## 问题现象
- ✅ 代码已更新：统一进度条颜色逻辑已实现
- ❌ 页面显示旧颜色：浏览器使用了缓存的旧 JavaScript 文件

---

## 解决方案（任选其一）

### 方案1：硬刷新（推荐，最快）
**macOS**: `Cmd + Shift + R`  
**Windows/Linux**: `Ctrl + Shift + R`

### 方案2：开发者工具清除缓存
1. 打开开发者工具（F12 或右键 → 检查）
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"

### 方案3：清除所有缓存
**Chrome/Edge**:
1. 打开 `chrome://settings/clearBrowserData`
2. 选择"缓存的图片和文件"
3. 点击"清除数据"

**Safari**:
1. 菜单栏 → 开发 → 清空缓存
2. 或者 `Cmd + Option + E`

---

## 验证方法
清除缓存后，检查订单卡片进度条颜色：

| 距离交期天数 | 预期颜色 | 说明 |
|------------|---------|------|
| > 3天 | **绿色** | 正常生产 |
| 2-3天 | **黄色** | 临近交期 |
| 0-1天 | **浅红色** | 警告 |
| 延期1-3天 | **红色** | 危险 |
| 延期>3天 | **深红色** | 严重延期 |

---

## 技术原因
Vite 构建时会对 JavaScript 文件生成哈希值（如 `index-abc123.js`），但浏览器可能会缓存旧版本。硬刷新可以强制重新下载最新文件。
