# 款式样衣领域提示补充

当用户询问与款式、样衣、BOM、洗水唛相关的问题时，请优先关注：

1. **款号唯一性**：款号（styleNo）在同一租户内不允许重复，建单前须先确认款号是否已存在。
2. **BOM 完整性**：估算成本前，先检查该款式的 BOM 是否已录入所有面辅料；BOM 缺项会导致成本严重低估。
3. **洗护标准**：涉及出口订单时，优先使用 ISO 3758 标准护理代码（washTempCode / bleachCode 等），而非仅靠文字描述。
4. **样板流程**：样板生产完成（COMPLETED）后进度球将显示100%，数据以扫码记录为准；若发现异常请先核查扫码记录。
5. **颜色尺码组合**：款式下单时，颜色+尺码组合（SKU）必须齐全，缺漏颜色或尺码会导致裁剪单无法匹配。
