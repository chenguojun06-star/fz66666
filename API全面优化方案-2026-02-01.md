### 1.2 主要问题

#### 🔴 问题1：查询端点过多（50+个）

**相同功能的不同端点**：


**✅ 优化方案：统一查询端点**

```java

// 方案2：保留单个查询，合并其他查询
```

**实施方法**：

```java
// 示例：生产订单Controller优化
@RestController
@RequestMapping("/api/production/order")
public class ProductionOrderController {
    
    // ❌ 删除这些端点
    // @GetMapping("/list-all")
    // @GetMapping("/by-order-no/{orderNo}")
    // @GetMapping("/by-status")
    
    // ✅ 统一为一个灵活的list端点
    @GetMapping("/list")
    public Result<IPage<ProductionOrder>> list(
        @RequestParam(required = false) String id,
        @RequestParam(required = false) String orderNo,
        @RequestParam(required = false) String status,
        @RequestParam(required = false) String styleNo,
        @RequestParam(defaultValue = "1") Integer page,
        @RequestParam(defaultValue = "10") Integer size
    ) {
        // 根据参数动态查询
        QueryWrapper<ProductionOrder> qw = new QueryWrapper<>();
        if (id != null) qw.eq("id", id);
        if (orderNo != null) qw.like("order_no", orderNo);
        if (status != null) qw.eq("status", status);
        if (styleNo != null) qw.eq("style_no", styleNo);
        
        IPage<ProductionOrder> result = service.page(
            new Page<>(page, size), qw
        );
        return Result.success(result);
    }
    
    // 保留单个详情查询（RESTful标准）
    @GetMapping("/{id}")
    public Result<ProductionOrder> getById(@PathVariable String id) {
        return Result.success(service.getById(id));
    }
}
```

## 🛡️ 四、风险控制

### 4.1 兼容性保证

**方案1：双端点过渡期（推荐）**

```java
// 保留旧端点3个月，添加@Deprecated
@Deprecated
@GetMapping("/list-all")
public Result<?> listAllDeprecated() {
    log.warn("使用了已废弃的端点 /list-all，请改用 /list");
    return list(null, null, null, 1, Integer.MAX_VALUE);
}

// 新端点
@GetMapping("/list")
public Result<?> list(...) { }
```

**方案2：前端统一适配层**

```typescript
// 创建API适配器，统一处理新旧端点
class APIAdapter {
  async getOrderList(params: any) {
    // 优先使用新端点
    try {
      return await api.get('/production/order/list', { params });
    } catch (e) {
      // 降级到旧端点
      if (params.orderNo) {
        return await api.get(`/production/order/by-order-no/${params.orderNo}`);
      }
      return await api.get('/production/order/list-all');
    }
  }
}
```
