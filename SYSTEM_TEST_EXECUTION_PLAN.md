## 一、当前系统功能概览

### PC端功能模块（9个主要模块）
1. **仪表盘** - 数据概览
2. **基础资料** - 款号、订单、资料中心、单价流程

## 六、数据一致性检查

### 关键数据点验证
1. **订单数量 = 裁剪数量 = 各环节完成数量总和**
2. **合格数量 + 次品数量 = 质检总数量**
3. **入库数量 = 合格数量**
4. **库存数量 = 入库数量 - 发货数量**
5. **工序金额 = 数量 × 单价**
6. **总结算金额 = 各工序金额之和**

### PC端与手机端数据同步验证
- [ ] 手机端扫码后，PC端实时更新
- [ ] PC端修改后，手机端能看到最新数据
- [ ] 质检结果在两端显示一致
- [ ] 统计数据在两端一致
