## 🏗️ 技术实现模式

### 1. 简单按钮（P1）
```tsx
// 原实现
```

**应用场景**: 1-2 个简单按钮，无复杂逻辑

---

### 2. 条件渲染（P2）

#### 模式 A：状态判断
```tsx
// OrderTransfer - 根据状态显示不同按钮组
render: (_, record) => {
  if (record.status === 'pending') {
    return <RowActions actions={[接受, 拒绝]} />;
  }
  if (record.status === 'rejected') {
    return <span>拒绝原因: {record.rejectReason}</span>;
  }
  return <span>-</span>;
}
```

#### 模式 B：动态构建数组
```tsx
// PayrollOperatorSummary - 根据审核状态
const actions = [
  { label: approved ? '已审核' : '审核', disabled: approved, ... }
];
if (approved) {
  actions.push({ label: '退回', danger: true, ... });
}
return <RowActions actions={actions} />;
```

#### 模式 C：条件返回 null
```tsx
// LoanHistoryDrawer - 仅借出状态显示
render: (_, record) => (
  record.status === 'borrowed' ? (
    <RowActions actions={[{ label: '归还', ... }]} />
  ) : null
)
```

1. **RowActions 组件能力**:
   - 支持 vertical 垂直布局

   - 集成 Dropdown（children 属性）
