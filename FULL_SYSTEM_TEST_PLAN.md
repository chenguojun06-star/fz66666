## 🔄 完整订单生命周期测试场景

### 场景1：标准订单流程
```
1. 创建订单 (PC端下单管理)
   ├─ 填写订单信息：订单号、款号、颜色、尺码、数量
   ├─ 设置交货日期
   └─ 保存订单

2. 物料采购 (手机端扫码)
   ├─ 扫描订单号
   ├─ 查看物料清单
   ├─ 更新到货数量
   └─ 完成采购领取

3. 裁剪环节 (手机端扫码)
   ├─ 扫描订单号或菲号
   ├─ 系统生成菲号 (PO-ST-颜色-尺码-数量-序号)
   ├─ 输入裁剪数量
   └─ 完成裁剪扫码

4. 缝制环节 (手机端扫码)
   ├─ 扫描菲号
   ├─ 输入缝制数量
   └─ 记录缝制工序

5. 车缝环节 (手机端扫码)
   ├─ 扫描菲号
   ├─ 输入车缝数量
   └─ 记录车缝工序

6. 大烫环节 (手机端扫码)
   ├─ 扫描菲号
   ├─ 输入大烫数量
   └─ 记录大烫工序

7. 质检环节
   a) 领取质检任务 (手机端扫码)
      ├─ 扫描菲号
      └─ 领取质检任务
   
   b) 处理质检任务 (手机端我的工作)
      ├─ 打开质检弹窗
      ├─ 查看菲号等详细信息
      ├─ 选择质检结果（合格/次品/返修完成）
      ├─ 如果是次品：
      │  ├─ 选择次品类型
      │  ├─ 上传次品图片
      │  ├─ 选择处理方式（返修/报废）
      │  └─ 填写备注
      └─ 提交质检结果

8. 包装环节 (手机端扫码)
   ├─ 扫描菲号
   ├─ 输入包装数量
   └─ 记录包装工序

9. 入库环节 (手机端扫码)
   ├─ 扫描菲号
   ├─ 选择入库仓库
   ├─ 输入入库数量
   └─ 完成入库

10. 发货结算 (PC端成品结算)
    ├─ 查看订单完成情况
    ├─ 生成结算单
    ├─ 审核确认
    └─ 完成订单

11. 财务对账 (PC端物料对账 + 成品结算)
    ├─ 物料对账单生成
    ├─ 成品结算单生成
    ├─ 审批付款
    └─ 完成对账
```

### 场景2：包含次品返修流程
```
1-6. 同标准流程

7. 质检发现次品
   ├─ 标记次品
   ├─ 上传次品图片
   ├─ 选择返修
   └─ 生成返修任务

8. 返修处理
   ├─ 返修完成
   └─ 重新质检

9. 质检合格
   └─ 继续后续流程

10-11. 同标准流程
```
