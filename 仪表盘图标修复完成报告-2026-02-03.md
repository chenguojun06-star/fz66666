1. **仪表盘图标消失**：质检统计概览的5个图标全部不显示（入库数、次品数、次品率、合格率、返修问题）

## 🔍 根本原因

### 问题1：CSS 变量未定义

**两个组件使用了不存在的 CSS 变量**：

#### MiniDataDashboard (质检统计)
```css
/* ❌ 错误：这些变量不存在 */
.mini-stat-icon--warehousing {
  background: var(--primary-color);      /* 未定义 */
}
.mini-stat-icon--repair {
  background: var(--primary-color-light); /* 未定义 */
}
.mini-stat-icon--defective {
  background: var(--warning-color);       /* 未定义 */
}
.mini-stat-icon--defect-rate {
  background: var(--error-color);         /* 未定义 */
}
.mini-stat-icon--qualified {
  background: var(--success-color);       /* 未定义 */
}
```

#### TopStats (顶部统计)
```typescript
// ❌ 错误：这些变量不存在
color: 'var(--primary-color-light)',
color: 'var(--success-color)',
```

**结果**：浏览器无法解析变量，图标背景变成透明/白色，看不见了。

### 问题2：旧 API 端点调用

**6个文件还在调用已废弃的 API**：
1. `FinishedSettlementContent.tsx` - `/finished-settlement/page`
2. `DashboardContent.tsx` - `/finished-settlement/page` (2处)
3. `MaterialInventory/index.tsx` - `/material/stock/page`
4. `PurchaseCreateForm.tsx` - `/material/stock/page`
