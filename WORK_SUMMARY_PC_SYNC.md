## 🚀 使用方法

```typescript
// 1. 导入
import { useSync } from '../../utils/syncManager';

// 2. 使用
useSync(
  'task-id',
  async () => api.get('/api/endpoint'),
  (newData, oldData) => {
    if (oldData !== null) setState(newData);
  },
  { interval: 30000, enabled: !loading, pauseOnHidden: true }
);
```

## ⚠️ 注意事项

1. **不要过度使用**：只在数据变化频繁的页面使用
2. **唯一任务ID**：每个页面使用不同的taskId
3. **自动清理**：组件卸载时会自动停止
4. **监控压力**：定期检查服务器请求量
