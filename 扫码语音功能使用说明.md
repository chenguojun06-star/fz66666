## 🔧 使用方法

### 在代码中播放语音

```javascript
// 导入语音管理器
const { voiceManager } = require('../../utils/voiceManager');

// 播放成功提示
voiceManager.play('success');

// 播放失败提示
voiceManager.play('error');

// 播放已被领取提示
voiceManager.play('claimed');
```

### 开关语音

```javascript
// 禁用语音
voiceManager.setEnabled(false);

// 启用语音
voiceManager.setEnabled(true);

// 获取当前状态
const isEnabled = voiceManager.getEnabled();
```

## 🔍 故障排查

### 听不到声音？

**检查清单**：
- [ ] 手机音量已开启
- [ ] 使用真机测试（非模拟器）
- [ ] 音频文件存在（`ls miniprogram/assets/audio/*.mp3`）
- [ ] 语音功能已启用（`voiceManager.getEnabled()` 返回 true）

**解决方法**：
```javascript
// 在微信开发者工具 Console 中测试
const { voiceManager } = require('../../utils/voiceManager');
voiceManager.play('success');
```

### 声音太小？

**方法 1：调高手机音量**（媒体音量，非铃声）

**方法 2：修改代码音量**
```javascript
// voiceManager.js
const VOICE_CONFIG = {
  volume: 2.0,  // 改为最大音量
};
```

**方法 3：重新生成音频**
编辑生成脚本，提高音量参数

### 声音卡顿或延迟？

**原因**：
- 网络问题（云端 TTS）
- 本地文件优先加载

**解决**：
代码已自动优先使用本地音频文件，延迟 < 100ms

### 文件路径错误？

检查路径是否正确：
```javascript
// 正确路径（相对于小程序根目录）
'/assets/audio/scan-success.mp3'

// 错误路径
'./assets/audio/scan-success.mp3'  // ❌ 不要加 ./
'miniprogram/assets/audio/scan-success.mp3'  // ❌ 不要加 miniprogram
```

## 📊 技术细节

### 音频规格

| 项目 | 参数 |
|------|------|
| 格式 | MP3 |
| 采样率 | 24000 Hz |
| 比特率 | 64 kbps |
| 声道 | 单声道 |
| 音质 | 清晰自然 |

### 语音库

使用 Google Text-to-Speech (gTTS)：
- 完全免费
- 无需 API key
- 支持中文（普通话）
- 音质自然流畅

### 播放机制

```
用户扫码
  ↓
触发回调
  ↓
voiceManager.play('success')
  ↓
优先使用本地音频文件
  ↓
InnerAudioContext 播放
  ↓
用户听到语音提示
```
