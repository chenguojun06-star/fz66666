   - `ironingStartTime`, `ironingEndTime`
   - `ironingOperatorName`, `ironingCompletionRate`
   - `packagingStartTime`, `packagingEndTime`
   - `packagingOperatorName`, `packagingCompletionRate`

2. **质量统计字段：**
   - `unqualifiedQuantity` (次品数)
   - `repairQuantity` (返修数)

3. **质检详情字段：**
   - `color`, `size`
   - `scanCode` (菲号)
   - `defectCategory` (次品类型)
   - `repairRemark` (处理方式)
   - `qualityOperatorName` (质检人员)

4. **采购字段：**
   - `expectedArrivalDate` (预计到货)
   - `actualArrivalDate` (实际到货)

5. **对账字段：**
   - `paidAmount` (已付金额)
   - `periodStartDate`, `periodEndDate` (对账周期)
   - `reconciliationOperatorName` (对账人)
   - `auditOperatorName` (审核人)

### 🔍 前端显示逻辑

如果后端字段暂时缺失，前端会显示：
- 数值类型：`0`
