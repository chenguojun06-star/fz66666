# 小程序管理员分配指南

## 🎯 问题：小程序看不到管理员功能

如果您的小程序个人中心显示"暂无权限信息"，需要在PC端后台为用户分配角色。

## 📋 解决方案

### 方法1：PC端后台分配角色（推荐）

#### 步骤1：登录PC端后台
打开浏览器访问：`http://localhost:5174`
- 使用管理员账号登录（如：admin / 密码）

#### 步骤2：进入用户管理
1. 点击左侧菜单"系统管理"
2. 点击"用户管理"

#### 步骤3：查找需要设置的用户
- 在用户列表中找到对应的员工账号
- 或者使用搜索功能查找

#### 步骤4：编辑用户角色
1. 点击用户行的"编辑"按钮
2. 在"角色"下拉框中选择角色：
   - **管理员** - 全部权限（推荐）
   - **主管** - 全部权限
   - **裁剪工** - 只看裁剪
   - **车缝工** - 只看车缝
   - **质检员** - 只看质检
   - **包装工** - 只看包装
   - **仓管员** - 只看入库
3. 点击"保存"

#### 步骤5：员工重新登录小程序
- 员工在小程序点击"退出登录"
- 重新登录
- 此时应该能看到角色和权限

### 方法2：创建管理员角色（如果没有）

#### 步骤1：创建角色
1. PC端后台 → 系统管理 → 角色管理
2. 点击"新增角色"
3. 填写信息：
   - 角色名称：`管理员`（必须包含"管理员"或"admin"关键字）
   - 角色代码：`admin`
   - 描述：系统管理员，拥有全部权限
   - 状态：启用
4. 点击"保存"

#### 步骤2：配置权限
1. 在角色列表找到"管理员"
2. 点击"权限配置"
3. 勾选所有权限模块
4. 点击"保存"

#### 步骤3：分配给用户
按照**方法1的步骤3-5**操作

## 🐛 常见问题排查

### Q1: 分配角色后小程序还是显示"暂无权限信息"
**解决方案**：
1. 确认用户在小程序已退出登录
2. 重新登录小程序
3. 清除小程序缓存：开发者工具 → 清缓存 → 全部清除

### Q2: PC端后台没有"角色管理"菜单
**解决方案**：
1. 确认当前登录账号是管理员
2. 刷新页面重新加载菜单
3. 检查账号权限配置

### Q3: 小程序登录后没有显示用户姓名
**可能原因**：
1. 用户表中的`name`字段为空
2. 后端登录接口没有返回用户信息

**解决方案**：
在PC端后台编辑用户，填写"姓名"字段

### Q4: 角色名称不包含关键字
**问题**：角色名称是"员工"、"普通用户"等，系统无法识别权限

**解决方案**：
修改角色名称，必须包含以下关键字之一：
- `管理员` 或 `admin` → 全部权限
- `主管` 或 `supervisor` → 全部权限
- `裁剪` → 裁剪工权限
- `车缝` 或 `缝制` → 车缝工权限
- `质检` → 质检员权限
- `包装` → 包装工权限
- `仓管` 或 `仓库` → 仓管员权限

## 💡 建议的角色配置

建议在PC端后台创建以下标准角色：

| 角色名称 | 角色代码 | 权限范围 | 小程序显示 |
|---------|---------|---------|-----------|
| 系统管理员 | admin | 全部 | 能看所有订单和功能 |
| 生产主管 | supervisor | 全部 | 能看所有订单和功能 |
| 裁剪工 | cutter | 裁剪 | 只看裁剪阶段订单 |
| 车缝工 | sewing | 车缝 | 只看车缝阶段订单 |
| 质检员 | quality | 质检 | 只看质检阶段订单 |
| 包装工 | packager | 包装 | 只看包装阶段订单 |
| 仓管员 | warehouse | 入库 | 只看入库阶段订单 |

## 🎯 完整操作流程示例

**场景**：为员工"张三"分配管理员权限

1. **PC端操作**（管理员账号）
   ```
   登录 http://localhost:5174
   ↓
   系统管理 → 用户管理
   ↓
   找到"张三"的账号
   ↓
   点击"编辑"
   ↓
   角色：选择"系统管理员"
   ↓
   点击"保存"
   ```

2. **小程序操作**（张三的账号）
   ```
   点击"退出登录"
   ↓
   重新输入账号密码登录
   ↓
   切换到"个人"标签页
   ↓
   查看是否显示：
   - 姓名：张三
   - 角色：系统管理员
   - 权限：[基础资料] [生产管理] 等
   ```

3. **验证权限**
   ```
   切换到"工作"标签页
   ↓
   应该能看到所有订单
   ↓
   切换到"扫码"标签页
   ↓
   应该能使用所有扫码类型
   ```
