## 流程2: 菲号扫码 (BUNDLE模式)

```
┌─────────────────────────────────────────────────────────────┐
│  用户扫描菲号: PO20260122001-黑色-01                        │
└────────────────┬──────────────────────────────────────────┘
                 │
                 ▼
    ┌──────────────────────────────────┐
    │  QRCodeParser.parse()            │
    │  识别为 BUNDLE 模式               │
    │  解析:                            │
    │  - orderNo: 'PO20260122001'       │
    │  - color: '黑色'                  │
    │  - batchNo: '01'                  │
    └──────────┬───────────────────────┘
               │
               ▼
    ┌──────────────────────────────────┐
    │  ScanHandler.handle()            │
    │  获取菲号详情                    │
    └──────────┬───────────────────────┘
               │
               ▼
    后端返回菲号信息:
    {
      bundleNo: 'PO20260122001-黑色-01',
      orderNo: 'PO20260122001',
      color: '黑色',
      sizeList: ['L', 'M'],
      quantity: 80,
      skuJson: [
        {color: '黑色', size: 'L'},
        {color: '黑色', size: 'M'}
      ]
    }
    │
    ▼
    SKUProcessor处理:
    - 不需要用户确认
    - 直接使用菲号中的数量
    - 所有尺码一起提交
    │
    ▼
    ┌──────────────────────────────────┐
    │  生成扫码请求:                   │
    │  [{                             │
    │    orderNo: '...',              │
    │    color: '黑色',               │
    │    size: 'L',                   │
    │    quantity: 80,                │
    │    bundleNo: 'PO...'           │
    │  },                             │
    │  {                             │
    │    orderNo: '...',              │
    │    color: '黑色',               │
    │    size: 'M',                   │
    │    quantity: 80,                │
    │    bundleNo: 'PO...'           │
    │  }]                             │
    └──────────┬───────────────────────┘
               │
               ▼
    提交给后端:
    - 关联菲号信息
    - 快速更新进度
    - 返回成功响应
               │
               ▼
    用户界面:
    ✓ 菲号 PO20260122001-黑色-01 已提交
    ✓ 数量: 80件
    → 下一步: 扫描下一菲号或其他订单
```

│ styleNo → FK t_style        │
│ styleName                   │
│ orderQuantity               │
│ completedQuantity           │
│ currentProcessName          │
│ orderDetails (JSON) ⬇      │
│ progressWorkflowJson        │

└──────────────┬──────────────┘
               │
               │ orderDetails JSON:
               │ [{
               │   color: '黑色',
               │   size: 'L',

               │   completedQty: 30
               │ }, ...]
               │
               ├──────────────────────────┐
               │                          │
               ▼                          ▼
    ┌──────────────────────┐  ┌──────────────────────┐
    │  t_scan_record       │  │  t_cutting_bundle    │
    ├──────────────────────┤  ├──────────────────────┤

    │ operatorId           │  └──────────────────────┘

## 关键对象定义

```
SKU {
  ┌────────────────────────────┐
  │ 识别字段 (唯一确定一个SKU) │
  ├────────────────────────────┤
  │ styleNo: 'ST001'           │
  │ color: '黑色'              │
  │ size: 'L'                  │
  │ orderNo: 'PO20260122001'   │
  └────────────────────────────┘
  
  ┌────────────────────────────┐
  │ 数量字段                    │
  ├────────────────────────────┤
  │ totalQuantity: 50          │
  │ completedQuantity: 30      │
  │ pendingQuantity: 20        │
  │ (= total - completed)      │
  └────────────────────────────┘
  
  ┌────────────────────────────┐
  │ 可选关联字段               │
  ├────────────────────────────┤
  │ bundleNo?: 'PO-黑色-01'   │
  └────────────────────────────┘
}

Order {
  ┌────────────────────────────┐
  │ 订单基本信息                │
  ├────────────────────────────┤
  │ orderNo: 'PO20260122001'   │
  │ styleNo: 'ST001'           │
  │ styleName: '连衣裙'        │
  └────────────────────────────┘
  
  ┌────────────────────────────┐
  │ SKU列表                    │
  ├────────────────────────────┤
  │ items: SKU[]               │
  │   - SKU (黑色, L)          │
  │   - SKU (黑色, M)          │
  │   - ...                    │
  └────────────────────────────┘
  
  ┌────────────────────────────┐
  │ 进度信息                    │
  ├────────────────────────────┤
  │ currentStage: '裁剪'       │
  │ progressWorkflow: {...}    │
  │ currentProcessName: '裁剪' │
  └────────────────────────────┘
}

Bundle {
  ┌────────────────────────────┐
  │ 菲号标识                    │
  ├────────────────────────────┤
  │ bundleNo: 'PO-黑色-01'    │
  │ orderNo: 'PO20260122001'   │
  │ color: '黑色' (单色)       │
  │ sizeList: ['L', 'M']       │
  │ quantity: 80               │
  └────────────────────────────┘
}
```
