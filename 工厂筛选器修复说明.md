# 工厂筛选器数据加载问题修复

## 🐛 问题描述
工厂筛选器下拉列表为空，无法显示工厂数据。

## 🔍 原因分析
API 响应数据结构解析错误：

**错误代码**：
```typescript
if (response.data?.code === 200) {
  setFactories(response.data.data.records || []);
}
```

**问题**：`api.get()` 已经自动解构了响应，实际结构应该是：
```typescript
response = {
  code: 200,
  data: {
    records: [...],
    total: 100
  }
}
```

## ✅ 修复方案
修正数据结构访问路径：

```typescript
if (response.code === 200) {
  setFactories(response.data.records || []);
}
```

## 📋 修改文件
- `frontend/src/modules/finance/pages/FinanceCenter/DashboardContent.tsx`（第136-143行）

## 🧪 验证方法
1. 刷新页面（Ctrl+R 或 Cmd+R）
2. 导航到：财务中心 → 数据看板
3. 点击"选择工厂"下拉框
4. 应该能看到系统中的所有工厂列表

## 📚 参考
参考了 `OrderManagement/index.tsx` 中的正确实现：
```typescript
const response = await api.get('/system/factory/list', { params: { page: 1, pageSize: 1000 } });
if (response.code === 200) {
  setFactories(response.data.records || []);
}
```
