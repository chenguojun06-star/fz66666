# 扫码语音提示音频文件说明

## 📁 音频文件列表

需要准备以下 3 个音频文件：

1. **scan-success.mp3** - 扫码成功提示音
   - 建议文本："扫码成功"
   - 时长：1-2 秒
   - 音调：温柔甜美
   - 大小：<20KB

2. **scan-error.mp3** - 扫码失败提示音
   - 建议文本："扫码失败，请重试"
   - 时长：1-2 秒
   - 音调：温柔提醒
   - 大小：<20KB

3. **scan-duplicate.mp3**（可选）- 重复扫码提示音
   - 建议文本："重复扫码"
   - 时长：1 秒
   - 音调：温柔
   - 大小：<15KB

**微软 Azure TTS**（音质最好）：

### 方案 2：使用 Python 脚本批量生成（开发者友好）

**安装依赖**：
```bash
pip install baidu-aip
```

**生成脚本**（auto-generate-voice.py）：
```python
from aip import AipSpeech

# 百度 AI 配置（需要注册获取）
APP_ID = 'your_app_id'
API_KEY = 'your_api_key'
SECRET_KEY = 'your_secret_key'

client = AipSpeech(APP_ID, API_KEY, SECRET_KEY)

# 生成语音
texts = {
    'scan-success.mp3': '扫码成功',
    'scan-error.mp3': '扫码失败，请重试',
    'scan-duplicate.mp3': '重复扫码',
}

for filename, text in texts.items():
    result = client.synthesis(
        text,
        'zh',      # 语言
        1,         # 客户端类型
        {
            'vol': 7,  # 音量（5-15）
            'pit': 6,  # 音调（0-15）
            'spd': 5,  # 语速（0-15）
            'per': 4,  # 发音人：4-度逍遥（温柔女声）
        }
    )
    
    if not isinstance(result, dict):
        with open(filename, 'wb') as f:
            f.write(result)
        print(f'✓ 生成成功: {filename}')
    else:
        print(f'✗ 生成失败: {filename}', result)
```

### 方案 3：录制真人语音（专业方案）

**设备要求**：
- 手机/电脑麦克风
- 录音软件（手机自带录音机、Audacity 等）

**录制步骤**：
1. 在安静环境录音
2. 用温柔甜美的声音说"扫码成功"
3. 导出为 MP3 格式
4. 使用 Audacity 剪辑（去除首尾空白）
5. 压缩音质（比特率 64kbps）

## 🔧 音频文件要求

**技术规格**：
- 格式：MP3（最佳兼容性）
- 采样率：44100 Hz 或 22050 Hz
- 比特率：64-128 kbps（平衡质量与大小）
- 声道：单声道或立体声
- 大小：单个文件 <30KB

**音质标准**：
- 清晰度：语音清晰，无噪音
- 音量：适中，不刺耳
- 时长：简短（1-2 秒）
- 音调：温柔甜美

## 💡 小提示

1. **音频文件命名不要改**，代码中已硬编码路径
2. **使用 64kbps 比特率**可显著减小文件大小
3. **建议使用单声道**（Mono）而非立体声
4. **录音环境要安静**，避免背景噪音
5. **可以加上背景音效**（如轻微的"叮"声）增加层次感
