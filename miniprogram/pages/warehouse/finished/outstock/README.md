### 📝 使用说明

1. **查看库存**: 搜索订单号或款号
2. **点击出库**: 点击订单右侧的"出库"按钮
3. **选择SKU**: 查看所有可用颜色和尺码
4. **输入数量**: 为需要出库的SKU输入数量
5. **快速填充**: 点击"MAX"按钮填充最大值
6. **查看汇总**: 实时显示总出库数量
7. **确认出库**: 点击"确认出库"完成

### ⚠️ 注意事项

1. **数量限制**: 出库数量不能超过可用库存
2. **至少一个**: 必须至少为一个SKU输入出库数量
3. **批量提交**: 系统会为每个SKU创建独立的出库记录
4. **备注信息**: 备注会自动附加SKU编码
