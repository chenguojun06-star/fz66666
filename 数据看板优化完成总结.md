### 2. 数值变化显示（已完成）

#### 优化前后对比

**优化前**：
```
较上月 ↑ 0.0%
```

**优化后**：
```
较上月 ↑ 15.3% (+¥12,345)  // 总金额
较上月 ↑ 8.5% (+450)       // 入库数量
较上月 ↑ 12.0% (+3)        // 订单数量
较上月 ↓ 5.2% (-8)         // 次品数量
```

#### 技术实现
```typescript
// 接口扩展 - 增加上期数值字段
interface StatCardData {
  totalAmount: number;
  prevTotalAmount: number;     // 新增
  totalAmountChange: number;
  
  warehousedCount: number;
  prevWarehousedCount: number; // 新增
  warehousedChange: number;
  
  orderCount: number;
  prevOrderCount: number;      // 新增
  orderChange: number;
  
  defectCount: number;
  prevDefectCount: number;     // 新增
  defectChange: number;
}

// 渲染函数优化
const renderChange = (
  value: number,        // 变化百分比
  currentValue: number, // 当期数值
  prevValue: number,    // 上期数值
  isMoney: boolean = false  // 是否为金额
) => {
  const isUp = value >= 0;
  const diff = currentValue - prevValue;
  const diffText = isMoney 
    ? `¥${Math.abs(diff).toLocaleString()}`
    : Math.abs(diff).toLocaleString();
  
  return (
    <span className={isUp ? styles.changeUp : styles.changeDown}>
      {isUp ? <CaretUpOutlined /> : <CaretDownOutlined />}
      {Math.abs(value).toFixed(1)}% ({isUp ? '+' : '-'}{diffText})
    </span>
  );
};

// 调用示例
{renderChange(
  statData.totalAmountChange,
  statData.totalAmount,
  statData.prevTotalAmount,
  true  // 金额类型
)}
```

## 📊 数据流程

### 筛选器数据流
```
页面初始化
  ↓
loadFactories() → /system/factory/list → 工厂下拉列表
  ↓
loadProcesses() → 固定数据 → 工序下拉列表
  ↓
用户选择工厂/工序
  ↓
触发 useEffect (监听 selectedFactory, selectedProcess)
  ↓
loadData() 携带筛选参数
  ├─ 当期数据：/finance/finished-settlement/page?factoryId=xxx&startDate=xxx&endDate=xxx
  └─ 上期数据：/finance/finished-settlement/page?factoryId=xxx&startDate=prev&endDate=prev
  ↓
计算统计指标 + 变化率 + 绝对差值
  ↓
更新 UI（4个统计卡片 + 趋势图 + 排名）
```

### 数值计算流程
```
获取当期数据 records[] 和上期数据 prevRecords[]
  ↓
计算当期统计
  ├─ totalAmount = Σ records.totalAmount
  ├─ warehousedCount = Σ records.warehousedQuantity
  ├─ orderCount = records.length
  └─ defectCount = Σ records.defectQuantity
  ↓
计算上期统计
  ├─ prevTotalAmount = Σ prevRecords.totalAmount
  ├─ prevWarehousedCount = Σ prevRecords.warehousedQuantity
  ├─ prevOrderCount = prevRecords.length
  └─ prevDefectCount = Σ prevRecords.defectQuantity
  ↓
计算变化率
  calcChange = (curr - prev) / prev × 100%
  ↓
计算绝对差值
  diff = curr - prev
  ↓
渲染
  百分比: ↑ 15.3%
  绝对值: (+¥12,345)
```

## 📖 使用说明

### 用户操作流程
1. 打开页面：财务中心 → 数据看板
2. 选择工厂：点击"选择工厂"下拉框，选择目标工厂
3. 选择工序：点击"选择工序"下拉框，选择目标工序
4. 查看数据：统计卡片和趋势图自动刷新
5. 清空筛选：点击选择框的清空按钮(×)，恢复全部数据

### 数据解读
- **百分比**：当前周期相比上一周期的变化百分比
- **绝对值**：当前周期相比上一周期的具体数值变化
- **箭头**：
  - 绿色 ↑：上涨
  - 红色 ↓：下降
- **格式**：
  - 总金额：显示 ¥ 符号
  - 数量：显示纯数字
