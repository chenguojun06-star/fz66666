## 技术亮点总结

### 1. 条件渲染模式

#### 模式 A：条件显示不同按钮组
```tsx
// OrderTransfer
render: (_, record) => {
  if (record.status === 'pending') {
    return <RowActions actions={[接受, 拒绝]} />;
  }
  if (record.status === 'rejected') {
    return <span>拒绝原因: {record.rejectReason}</span>;
  }
  return <span>-</span>;
}
```

#### 模式 B：动态构建 actions 数组
```tsx
// PayrollOperatorSummary
const actions = [
  { label: approved ? '已审核' : '审核', disabled: approved, ... }
];
if (approved) {
  actions.push({ label: '退回', danger: true, ... });
}
return <RowActions actions={actions} />;
```

#### 模式 C：条件返回 null
```tsx
// LoanHistoryDrawer
render: (_, record) => (
  record.status === 'borrowed' ? (
    <RowActions actions={[归还]} />
  ) : null
)
```

### 2. 垂直布局应用
```tsx
// MaterialInventory - 5个按钮垂直排列
<RowActions
  vertical  // 关键：启用垂直布局
  actions={[采购指令, 入库, 出库, 打印, 详情]}
/>
```

### 3. Tooltip 集成
```tsx
// MaterialInventory
actions={[
  ...,
  {
    label: '详情',
    tooltip: '查看最近出入库记录',  // RowActions 自动处理 Tooltip
    onClick: () => handleViewDetail(record)
  }
]}
```

### 4. 事件传播控制
```tsx
// OrderTable - 防止行点击事件
{
  label: '详情',
  onClick: (e) => {
    e.stopPropagation();  // 保留原有逻辑
    onRowClick?.(record);
  }
}
```
