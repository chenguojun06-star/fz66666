## 🔧 特殊场景文件（2 个）

### 1. NodeDetailModal.tsx（详情弹窗）
**位置**: `components/common/NodeDetailModal.tsx`  
**场景**: 详情弹窗内的小表格（3 处）
- 裁剪记录 Tab
- 扫码记录 Tab
- 菲号明细 Tab

**配置**:
```typescript
pagination={{ 
  pageSize: 10, 
  size: 'small', 
  showTotal: (total) => `共 ${total} 条`,
  showSizeChanger: false  // 固定页大小，不提供切换
}}
```

**原因**: 弹窗内空间有限，数据量可控，使用固定页大小更简洁

### 2. StyleBomTab.tsx（面辅料选择器）
**位置**: `basic/pages/StyleInfo/components/StyleBomTab.tsx`  
**场景**: 添加面辅料时的材料选择弹窗

**配置**:
```typescript
pagination={{ 
  current: materialPage,
  pageSize: 10,
  total: materialTotal,
  showTotal: (total) => `共 ${total} 条`,
  onChange: (p) => fetchMaterials(p, materialKeyword),
  showSizeChanger: false  // 选择器场景，固定页大小
}}
```

**原因**: 选择器弹窗，数据选择为主，固定页大小减少干扰
