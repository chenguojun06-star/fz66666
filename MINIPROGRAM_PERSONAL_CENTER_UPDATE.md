# 小程序个人中心新增功能说明

## 🆕 新增功能

在小程序个人中心页面添加了以下两个功能：

### 1. 在线人数统计
- **显示位置**：个人中心 → 系统信息区域
- **功能说明**：实时显示当前在线用户数量（最近10分钟内登录的用户）
- **更新方式**：
  - 页面进入时自动加载
  - 下拉刷新时更新
  - 点击"刷新"按钮时更新

### 2. 我的权限
- **显示位置**：个人中心 → 我的权限区域
- **功能说明**：显示当前用户被分配的所有权限名称
- **权限来源**：根据用户的角色ID从后台获取权限列表
- **显示方式**：以标签形式展示所有权限名称

## 🔧 技术实现

### 前端文件修改
1. **miniprogram/pages/admin/index.js**
   - 添加 `loadingSystemInfo`、`onlineCount`、`permissions` 数据字段
   - 新增 `loadSystemInfo()` 方法加载系统信息
   - 新增 `extractPermissionNames()` 方法提取权限名称
   - 修改 `refreshAll()` 包含系统信息刷新

2. **miniprogram/pages/admin/index.wxml**
   - 添加"系统信息"区域显示在线人数
   - 添加"我的权限"区域显示权限标签

3. **miniprogram/pages/admin/index.wxss**
   - 添加 `.system-info` 样式
   - 添加 `.info-item`、`.info-label`、`.info-value` 样式
   - 添加 `.permissions-list`、`.permission-item` 样式

4. **miniprogram/utils/api.js**
   - 添加 `getOnlineCount()` 接口调用（后端已有）

### 后端接口（已存在）
- **在线人数接口**：`GET /api/system/user/online-count`
  - 返回最近10分钟内登录成功的不重复用户数
  - 已实现，无需修改

- **角色权限接口**：`GET /api/system/role/{roleId}/permission-ids`
  - 返回指定角色的所有权限ID列表
  - 已实现，无需修改

- **权限树接口**：`GET /api/system/permission/tree`
  - 返回完整的权限树结构
  - 已实现，无需修改

## 📊 数据流程

### 在线人数
```
小程序页面加载
    ↓
调用 api.system.getOnlineCount()
    ↓
后端查询最近10分钟登录记录
    ↓
统计不重复用户数
    ↓
返回数字（如：5）
    ↓
页面显示"5人"
```

### 我的权限
```
小程序页面加载
    ↓
获取用户角色ID（从 userInfo）
    ↓
调用 api.system.getRolePermissionIds(roleId)
    ↓
获取权限ID列表（如：[1, 2, 5, 8]）
    ↓
调用 api.system.getPermissionTree()
    ↓
获取完整权限树
    ↓
遍历权限树，提取对应ID的权限名称
    ↓
页面显示权限标签（如："裁剪任务"、"裁剪单"）
```

## 🐛 常见问题

### Q1: 在线人数显示为0
**A:** 检查以下几点：
1. 确认后端服务正常运行
2. 确认数据库中有最近10分钟的登录记录
3. 检查后端日志是否有错误

### Q2: 我的权限显示"暂无权限信息"
**A:** 可能原因：
1. 用户未分配角色（roleId为空）
2. 角色未配置权限
3. 权限数据加载失败（检查网络）

### Q3: 权限名称显示为空
**A:** 检查后端权限表中是否正确配置了权限名称（name或permissionName字段）

## 📝 注意事项

1. **在线人数统计规则**：
   - 统计最近10分钟内登录成功的不重复用户
   - 不是实时在线人数，而是活跃用户数
   - 同一用户多次登录只计数一次

2. **权限显示**：
   - 只显示权限名称，不显示权限代码
   - 如果权限树层级复杂，会显示所有层级的权限
   - 建议在后台为权限配置简短易懂的名称

3. **性能优化**：
   - 权限数据在页面加载时获取一次
   - 刷新时重新获取最新数据
   - 权限树数据较大时可能加载较慢

4. **权限控制**：
   - 所有登录用户都可以查看在线人数
   - 只能查看自己的权限，不能查看其他用户的权限
