## 技术要点

### RowActions 组件 API 使用
```tsx
<RowActions
  actions={[
    {
      label: '按钮文本',
      icon: <Icon />,          // 可选
      type: 'primary',         // 可选
      danger: true,            // 可选
      disabled: condition,     // 可选
      onClick: () => handler()
    }
  ]}
/>
```

### 常见模式转换

| 原实现 | 新实现 |
|--------|--------|
| `<Button danger type="link">删除</Button>` | `{ label: '删除', danger: true }` |
| `<Button icon={<EditOutlined />}>编辑</Button>` | `{ label: '编辑', icon: <EditOutlined /> }` |
| `<Button type="primary">出库</Button>` | `{ label: '出库', type: 'primary' }` |
| `<a onClick={handler}>详情</a>` | `{ label: '详情', onClick: handler }` |
