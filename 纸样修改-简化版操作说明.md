# 纸样修改功能 - 操作说明

## ✅ 已完成 - 简化版实现

**更新时间**: 2026-01-31 14:17  
**实现方式**: 在款号列表操作列直接添加"纸样修改"按钮

## 🎯 如何使用

### 第一步：进入资料中心
1. 在左侧菜单点击"资料中心"
2. 您会看到款号列表

### 第二步：找到需要修改的款号
在列表中找到需要记录纸样修改的款号（如：HHYO0322）

### 第三步：点击"纸样修改"按钮 ⭐
在操作列中，您会看到这些按钮：
- 👁️ 查看
- ✏️ 编辑
- **📄 纸样修改** ← 点击这个！
- 🖨️ 打印
- 📥 下载

### 第四步：填写修改信息
弹窗会显示以下字段：

```
纸样修改记录 - HHYO0322
├─ 款号: HHYO0322 (自动填充，不可修改)
├─ 修改类型: [必填] 
│   └─ 选项：小改 / 大改 / 紧急修改
├─ 修改日期: [可选] 选择日期
├─ 修改原因: [必填] 
│   └─ 例如：客户要求调整版型
├─ 修改内容: [必填]
│   └─ 详细描述需要修改的地方
├─ 纸样师傅: [可选] 输入师傅姓名
├─ 预计完成日期: [可选] 选择日期
└─ 备注: [可选] 其他说明
```

### 第五步：保存
点击"保存"按钮，系统会自动：
- ✅ 记录维护人（当前登录用户）
- ✅ 记录维护时间（当前时间）
- ✅ 生成版本号（V1.0, V1.1 ...）
- ✅ 保存到数据库

## 📝 使用示例

### 示例1: 记录一个小改
```
款号: HHYO0322
修改类型: 小改
修改日期: 2026-01-31
修改原因: 客户反馈袖长偏短
修改内容: 袖长增加2cm，从55cm改为57cm
纸样师傅: 张师傅
预计完成日期: 2026-02-01
备注: 需要同步修改裁剪单
```

### 示例2: 紧急修改
```
款号: HHYO0322
修改类型: 紧急修改
修改日期: 2026-01-31
修改原因: 面料缩水率超出预期
修改内容: 整体放大5%，重新制版
纸样师傅: 李师傅
预计完成日期: 2026-01-31
备注: 今日必须完成
```

## 🔍 查看修改记录

保存后，可以通过以下方式查看记录：
1. 访问完整的纸样修改记录管理页面：http://localhost:5173/basic/pattern-revision
2. 或者在数据库查看：`SELECT * FROM t_pattern_revision`

## 📊 数据存储

所有记录保存在 `t_pattern_revision` 表中，包含：
- 款号信息
- 修改类型、原因、内容
- 维护人和维护时间（自动填充）
- 版本号（自动生成）
- 预计完成日期
- 纸样师傅

## 🔧 故障排查

### 问题1: 看不到"纸样修改"按钮
**解决**: 刷新页面，确认前端服务已更新

### 问题2: 点击保存后提示错误
**解决**: 
1. 确认后端服务已启动
2. 检查 `/production/pattern-revision` API是否正常
3. 查看浏览器控制台错误信息

### 问题3: 保存成功但找不到记录
**解决**: 
访问纸样修改记录管理页面查看：
http://localhost:5173/basic/pattern-revision

或在数据库查询：
```sql
SELECT * FROM t_pattern_revision ORDER BY create_time DESC LIMIT 10;
```
