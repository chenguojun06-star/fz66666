## 🎯 关键 CSS 代码示例

### 主页面输入框（40px 高度，正方形）

```css
/* 全站输入/选择/日期统一高度与正方形（排除登录和仪表盘） */
:where(.ant-input, .ant-input-affix-wrapper, .ant-input-number, .ant-select-selector, .ant-picker):not([class*="login"]):not([class*="dashboard"]) {
  height: 40px;
  border-radius: 0 !important;
  border-color: color-mix(in srgb, var(--color-border) 55%, var(--color-text-primary) 45%);
  background: var(--color-bg-base);
  transition: all var(--duration-base) var(--ease-smooth);
}
```

### 文字居中效果

```css
/* 输入框内容居中 */
:where(.ant-input, .ant-input-affix-wrapper input, .ant-input-number-input, .ant-picker-input > input):not([class*="login"]):not([class*="dashboard"]) {
  text-align: center;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* 选择器内容居中 */
:where(.ant-select-single .ant-select-selector .ant-select-selection-item,
.ant-select-single .ant-select-selector .ant-select-selection-placeholder):not([class*="login"]):not([class*="dashboard"]) {
  text-align: center;
  justify-content: center;
}
```

### 按钮保持胶囊形

/* 按钮基础高度与胶囊统一 */

  font-size: 14px !important;

## 📝 注意事项

### 1. 样式优先级

所有修改使用 `!important` 确保全局生效，覆盖 Ant Design 默认样式。

### 2. 排除登录和仪表盘

使用 CSS 选择器 `:not([class*="login"]):not([class*="dashboard"])` 排除，确保这两个页面保持原样式。

### 3. 文字居中实现

使用 `display: flex` + `align-items: center` + `justify-content: center` + `text-align: center` 四种方式组合，确保在所有浏览器中完全居中。

### 4. 按钮样式不受影响

按钮样式定义在独立文件 `button-override.css` 中，使用 `var(--radius-pill)` 保持胶囊形，不受本次改造影响。

### 5. 搜索框特殊处理

搜索框的**输入框部分**改为正方形，但**搜索按钮**保持胶囊形，两者视觉上分离。
