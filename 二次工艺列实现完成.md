### 🔧 技术细节

#### 检测逻辑（优先级）
```typescript
// 1. 检查扫码记录生成的时间数据（最准确）
if (record.secondaryProcessStartTime || record.secondaryProcessEndTime) {
  return true; // 有数据，显示彩色
}

// 2. 检查工序配置
if (record.progressNodeUnitPrices?.some(n => 
  n.name.includes('二次工艺') || n.name.includes('二次')
)) {
  return true; // 有配置，显示彩色
}

// 3. 都没有
return false; // 显示灰色占位
```

#### 渲染差异
```tsx
// 无数据：灰色占位
<div style={{ opacity: 0.4 }}>
  <div style={{ 
    background: '#e8e8e8',  // 灰色背景
    height: '12px',
    borderRadius: '6px'
  }} />
  <span style={{ color: '#999' }}>-</span>
</div>

// 有数据：彩色进度条
<div 
  style={{ cursor: 'pointer' }}
  onClick={() => openProcessDetail(record, 'secondaryProcess')}
>
  <LiquidProgressBar percent={rate} height={12} />
  <span>{rate}%</span>
</div>
```
