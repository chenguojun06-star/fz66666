# 样板生产功能完整测试指南

## 🎯 功能概述

完整的样板生产管理流程：

1. **自动创建** - 样衣开发下单时自动创建样板生产记录（状态：待领取）
2. **领取样板** - 纸样师傅领取任务（待领取 → 进行中）
3. **更新进度** - 纸样师傅更新 6 个工序的进度百分比
4. **自动完成** - 所有工序达到 100% 时自动标记为已完成

---

## 📊 数据完整性检查

### 检查自动创建
```sql
-- 统计自动创建的样板生产记录
SELECT 
  COUNT(*) as total,
  SUM(CASE WHEN status = 'PENDING' THEN 1 ELSE 0 END) as pending,
  SUM(CASE WHEN status = 'IN_PROGRESS' THEN 1 ELSE 0 END) as in_progress,
  SUM(CASE WHEN status = 'COMPLETED' THEN 1 ELSE 0 END) as completed
FROM t_pattern_production;
```

### 检查数据一致性
```sql
-- 验证 styleId 关联
SELECT 
  pp.id,
  pp.style_no,
  pp.status,
  si.style_no as style_info_no,
  si.style_name
FROM t_pattern_production pp
LEFT JOIN t_style_info si ON pp.style_id = si.id
WHERE pp.delete_flag = 0
LIMIT 10;
```

### 检查进度 JSON 格式
```sql
-- 验证 progress_nodes JSON 格式
SELECT 
  id,
  style_no,
  progress_nodes,
  JSON_VALID(progress_nodes) as is_valid_json
FROM t_pattern_production
WHERE delete_flag = 0;
```

## 🐛 常见问题排查

### 问题1: 领取后状态没变

**可能原因**：
- 前端没有刷新
- 后端事务未提交
- 数据库连接问题

**排查步骤**：
1. 检查 Network 请求是否成功（200）
2. 数据库直接查询状态
3. 查看后端日志是否有异常

---

### 问题2: 进度更新后液体动画没变

**可能原因**：
- 前端没有重新加载数据
- `progressNodes` JSON 解析失败
- 组件 props 未更新

**排查步骤**：
1. 打开 React DevTools 查看组件 state
2. 检查 `loadData()` 是否被调用
3. Console 查看是否有 JSON 解析错误

---

### 问题3: 自动完成没触发

**可能原因**：
- 某个工序进度 < 100
- 后端逻辑判断错误
- 数据库更新失败

**排查步骤**：
```sql
-- 检查所有工序是否都是100
SELECT 
  id, 
  style_no, 
  progress_nodes,
  status,
  complete_time
FROM t_pattern_production
WHERE id = 'YOUR_ID';

-- 手动解析JSON检查
SELECT 
  JSON_EXTRACT(progress_nodes, '$.cutting') as cutting,
  JSON_EXTRACT(progress_nodes, '$.sewing') as sewing,
  JSON_EXTRACT(progress_nodes, '$.ironing') as ironing,
  JSON_EXTRACT(progress_nodes, '$.quality') as quality,
  JSON_EXTRACT(progress_nodes, '$.secondary') as secondary,
  JSON_EXTRACT(progress_nodes, '$.packaging') as packaging
FROM t_pattern_production
WHERE id = 'YOUR_ID';
```
