# 样衣开发-二次工艺标签页 - 快速测试指南

## 📍 功能位置

**样衣管理 → 样衣开发 → [选择任意款式] → 二次工艺标签页**

位置：在"生产制单"和"样衣生产"标签之间

## ✅ 测试步骤

### 1. 进入页面
1. 启动前端：`cd frontend && npm run dev`
2. 登录系统
3. 点击左侧菜单：**样衣管理 → 样衣开发**
4. 点击任意一个款式记录，进入详情页
5. 点击底部标签页：**二次工艺**

### 2. 功能测试

#### 新建工艺
1. 点击"新建工艺"按钮
2. 填写表单：
   - 工艺类型：选择"刺绣"
   - 工艺名称：输入"胸前刺绣"
   - 数量：输入 500
   - 单价：输入 2.5
   - 总价：自动计算为 1250
   - 加工厂：输入"鸿达刺绣厂"
   - 状态：选择"处理中"
   - 备注：输入"需使用金线"
3. 点击"保存"

#### 查看列表
- 表格显示所有二次工艺记录
- 列：工艺类型、工艺名称、数量、单价、总价、加工厂、状态、创建时间、操作

#### 编辑工艺
1. 点击记录的"编辑"按钮
2. 修改数量或单价
3. 总价自动重新计算
4. 点击"保存"

#### 查看详情
- 点击"查看"按钮
- 弹窗显示完整信息（只读模式）

#### 删除工艺
- 点击"删除"按钮
- 确认删除对话框
- 点击"确定"完成删除

## 📊 工艺类型

系统支持以下工艺类型：
- 🧵 **刺绣** - embroidery
- 🎨 **印花** - printing
- 💧 **水洗** - washing
- 🌈 **染色** - dyeing
- 🔥 **烫印** - ironing
- 📏 **打褶** - pleating
- 💎 **钉珠** - beading
- ⚙️ **其他** - other

## 🎯 状态说明

- **待处理** (pending) - 灰色标签
- **处理中** (processing) - 蓝色标签
- **已完成** (completed) - 绿色标签
- **已取消** (cancelled) - 红色标签

## 💡 功能特点

### 自动计算
- 输入数量和单价后，总价自动计算
- 公式：总价 = 数量 × 单价

### 数据格式
- 单价：保留2位小数，带¥符号
- 总价：蓝色加粗显示，带¥符号
- 数量：千分位格式化显示

### 权限控制
- 只读模式：隐藏"新建工艺"按钮和操作列
- 编辑模式：显示完整功能

## 🔧 调试提示

### 检查组件加载
打开浏览器控制台，查看：

- 网络请求是否发送（会看到404，因为后端未开发）

1. **标签页不显示**：检查是否保存了基础信息（需要有 styleId）
2. **保存失败**：正常现象，后端API未实现
3. **数据不显示**：正常现象，数据库中暂无数据
