### 2. ✅ 较上周/上月显示具体数值

**优化前**：
```
较上月 ↑ 15.3%
```

**优化后**：
```
较上月 ↑ 15.3% (+¥12,345)  // 总金额卡片
较上月 ↑ 8.5% (+450)       // 入库数量卡片
较上月 ↑ 12.0% (+3)        // 订单数量卡片
较上月 ↓ 5.2% (-8)         // 次品数量卡片
```

**实现逻辑**：
1. 增加 `prevTotalAmount`、`prevWarehousedCount` 等字段记录上期数值
2. 修改 `renderChange` 函数，计算并显示绝对差值
3. 金额类数据显示货币符号 ¥，数量类数据显示纯数字

**代码示例**：
```tsx
const renderChange = (
  value: number,        // 变化百分比
  currentValue: number, // 当期数值
  prevValue: number,    // 上期数值
  isMoney: boolean = false
) => {
  const isUp = value >= 0;
  const diff = currentValue - prevValue;
  const diffText = isMoney 
    ? `¥${Math.abs(diff).toLocaleString()}`
    : Math.abs(diff).toLocaleString();
  
  return (
    <span className={isUp ? styles.changeUp : styles.changeDown}>
      {isUp ? <CaretUpOutlined /> : <CaretDownOutlined />}
      {Math.abs(value).toFixed(1)}% ({isUp ? '+' : '-'}{diffText})
    </span>
  );
};
```

### 统计数据刷新
```
选择筛选条件 → loadData() 
  ├─ 构建查询参数 (factoryId, startDate, endDate)
  ├─ 获取当期数据 → /finance/finished-settlement/page
  ├─ 获取上期数据 → /finance/finished-settlement/page (上周/上月时间范围)
  ├─ 计算统计指标（总金额、入库量、订单数、次品数）
  ├─ 计算变化率和绝对差值
  └─ 更新 4 个统计卡片 + 趋势图 + 排名
```

### 4. 缓存优化
使用 localStorage 记住用户上次选择的筛选条件：
```tsx
useEffect(() => {
  const saved = localStorage.getItem('dashboard-filters');
  if (saved) {
    const { factory, process } = JSON.parse(saved);
    setSelectedFactory(factory);
    setSelectedProcess(process);
  }
}, []);

useEffect(() => {
  localStorage.setItem('dashboard-filters', JSON.stringify({
    factory: selectedFactory,
    process: selectedProcess
  }));
}, [selectedFactory, selectedProcess]);
```
