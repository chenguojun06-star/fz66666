# 词典自动收录功能说明

## 📚 功能概述

系统现在支持**智能词典自动收录功能**：当用户在任何输入框中输入新的词汇时，系统会自动检测并将其添加到对应的词典类型中，无需手动维护词典数据。

## 🎯 工作原理

### 1. 自动检测与收录流程

```
用户输入 → 失焦(onBlur) → 防抖(1秒) → 检查是否已存在 → 不存在则自动添加到词典
```

### 2. 智能去重机制

- **本地缓存**：已收录的词汇会缓存在内存中，避免重复检查
- **后端验证**：首次输入时会调用 API 检查词典中是否已存在
- **静默收录**：默认静默模式，不会打扰用户操作

### 3. 防抖保护

- 输入后 **1秒** 内如果继续修改，会重置计时器
- 避免频繁的 API 调用，提升性能

## 🔧 技术实现

### 核心 Hook：`useAutoCollectDict`

位置：`frontend/src/hooks/useAutoCollectDict.ts`

```typescript
const { collectWord } = useAutoCollectDict({
  dictType: 'color',      // 词典类型
  enabled: true,          // 是否启用
  silent: true,           // 静默模式（不显示提示）
  debounceMs: 1000       // 防抖时间（毫秒）
});

// 在输入框失焦时调用
<Input onBlur={(e) => collectWord(e.target.value)} />
```

### 通用组件：`AutoCollectInput`

位置：`frontend/src/components/common/AutoCollectInput.tsx`

```typescript
<AutoCollectInput
  dictType="color"       // 指定词典类型
  autoCollect={true}     // 启用自动收录
  collectOnBlur={true}   // 失焦时收录
  collectOnChange={false} // 输入时收录（默认关闭）
  placeholder="输入颜色"
/>
```

## 📖 使用示例


## 🔍 数据格式

### 自动收录的数据结构

```json
{
  "dictType": "color",
  "dictCode": "COL_ZHZH_123456",
  "dictLabel": "酒红色",
  "sortOrder": 11,
  "remark": "自动收录"
}
```

**字段说明**：
- `dictType`：词典类型（color, size, fabric_type 等）
- `dictCode`：自动生成的唯一编码（前缀 + 标识 + 时间戳）
- `dictLabel`：用户输入的词汇
- `sortOrder`：自动排序（当前最大值 + 1）
- `remark`：标记为"自动收录"，便于区分

### 编码生成规则

```
格式：{类型前缀}_{标识}_{时间戳后6位}

示例：
- color + "酒红色" → COL_ZHZH_123456
- size + "3XL" → SIZ_3XL_123457
- fabric_type + "真丝" → FAB_ZHZH_123458
```

## ⚙️ 配置选项

### Hook 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| dictType | string | 必填 | 词典类型（参考下方类型列表） |
| enabled | boolean | true | 是否启用自动收录 |
| silent | boolean | true | 是否静默收录（不显示提示） |
| debounceMs | number | 500 | 防抖时间（毫秒） |

## 🛡️ 安全机制

### 1. 去重保护
- 输入前检查词典是否已存在
- 本地缓存已收录的词汇
- 避免重复添加相同词汇

### 2. 错误处理
- API 失败时静默处理，不影响用户操作
- 网络错误时缓存到本地，待恢复后同步

### 3. 性能优化
- 防抖机制减少 API 调用
- 本地缓存减少重复检查
- 异步处理不阻塞 UI

## 📊 后端 API 要求

### 1. 检查词汇是否存在

```
GET /api/system/dict/list?dictType={type}&dictLabel={label}&pageSize=1
```

### 2. 添加新词汇

```
POST /api/system/dict
Content-Type: application/json

{
  "dictType": "color",
  "dictCode": "COL_ZHZH_123456",
  "dictLabel": "酒红色",
  "sortOrder": 11,
  "remark": "自动收录"
}
```

### 3. 获取最大排序号

```
GET /api/system/dict/list?dictType={type}&pageSize=1&sortField=sort_order&sortOrder=desc
```
