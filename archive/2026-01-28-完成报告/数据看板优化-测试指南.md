## 🚀 启动环境

### 1. 确保后端运行
```bash

```

```bash
lsof -i :5173 || (cd frontend && npm run dev)
```

```
http://localhost:5173
```

## 🐛 常见问题排查

### 问题1：工厂列表为空
**检查**：
```bash
# 查看工厂数据
docker exec fashion-mysql-simple mysql -uroot -pchangeme fashion_supplychain \
  -e "SELECT id, factory_name FROM t_factory LIMIT 5;"
```

**解决**：如果数据库为空，需要先添加工厂数据

### 问题2：数据不刷新
**检查**：
- 打开浏览器控制台（F12）
- 查看 Network 标签
- 选择筛选器时是否有新的 API 请求
- 请求参数是否包含 `factoryId`

### 问题3：显示错误或异常
**检查**：
- 浏览器控制台是否有报错
- 后端日志：`tail -f backend/logs/spring.log`
- 检查 API 响应是否正常

### 问题4：数值变化不显示
**检查**：
- 确认是否有历史数据（上周/上月）
- 如果是新系统，可能没有对比数据
- 在 Network 标签查看两个 API 请求（当期+上期）
