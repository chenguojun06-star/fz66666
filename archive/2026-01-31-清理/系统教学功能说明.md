**创建日期**: 2026-01-28

## 📋 功能概述

**系统教学中心**是一个内置的操作指南平台,为用户提供从入门到精通的完整学习路径。通过**分步骤教程**、**常见问题解答**、**视频演示**等多种形式,帮助新用户快速上手系统操作。

### 核心价值
- ✅ **降低培训成本**: 用户可自助学习,减少人工培训时间
- ✅ **提升操作效率**: 标准化操作流程,避免错误操作
- ✅ **知识沉淀**: 将业务经验转化为可复用的教程
- ✅ **持续更新**: 系统升级时同步更新教程内容

## 🎯 功能特性

### 1. 分类教程体系
教程按业务模块分类,支持快速定位:
- 🚀 **生产管理** (4个教程)
  - 生产订单创建完整流程
  - 裁剪单生成与菲号管理
  - 质检入库完整流程
  - 对账与财务结算

- 📦 **仓储管理** (2个教程)
  - 面辅料出入库管理
  - 成品出库发货流程

- 📱 **小程序操作** (1个教程)
  - 小程序扫码工序操作

- ⚙️ **系统设置** (1个教程)
  - 人员与权限管理

### 2. 多维度筛选
- **分类筛选**: 点击分类按钮快速过滤
- **关键词搜索**: 支持搜索标题、标签、步骤内容
- **难度标签**: 入门 / 进阶 / 高级,适配不同用户水平

### 3. 结构化教程内容
每个教程包含3个Tab页:

#### 📖 操作步骤
- **分步展示**: 使用Steps组件,清晰展示操作流程
- **详细说明**: 每步包含标题、描述、操作技巧
- **温馨提示**: 高亮显示注意事项和最佳实践
- **截图示意**: 支持插入操作截图(未来版本)

#### ❓ 常见问题
- **问答形式**: Collapse折叠面板,方便快速查找
- **问题分类**: 按实际用户反馈整理高频问题
- **解决方案**: 提供明确的处理步骤

#### 🎬 视频教程
- **视频链接**: 预留视频播放功能(开发中)
- **时长标注**: 方便用户规划学习时间
- **多端支持**: 支持PC和移动端播放

### 4. 智能推荐
- **新手指引**: 首页显示建议学习路径
- **关联推荐**: 根据当前教程推荐相关内容(未来版本)
- **学习进度**: 记录已学习教程,推荐下一步(未来版本)

## 📚 已创建教程列表

### 生产管理模块

#### 1. 生产订单创建完整流程
- **难度**: 入门
- **时长**: 5分钟
- **核心步骤**:
  1. 进入订单管理页面
  2. 点击新建订单
  3. 选择款式
  4. 填写订单基本信息
  5. 配置SKU明细
  6. 保存并生成二维码
- **常见问题**: 3个
- **关键提示**:
  - 订单编号格式: PO + 8位日期 + 3位序号
  - SKU = 款号 + 颜色 + 尺码
  - 交货日期建议预留至少30天

#### 2. 裁剪单生成与菲号管理
- **难度**: 进阶
- **时长**: 8分钟
- **核心步骤**:
  1. 从订单创建裁剪单
  2. 配置裁剪明细
  3. 生成裁剪二维码
  4. 下发裁剪任务
- **常见问题**: 2个
- **关键提示**:
  - 菲号格式: 订单号 + 颜色 + 序号
  - 建议每个菲号控制在50-100件
  - 每个菲号有独立二维码

#### 3. 质检入库完整流程
- **难度**: 进阶
- **时长**: 7分钟
- **核心步骤**:
  1. 进入质检入库页面
  2. 创建质检记录
  3. 生成入库单
  4. 打印入库凭证
- **常见问题**: 2个

#### 4. 对账与财务结算
- **难度**: 高级
- **时长**: 10分钟
- **核心步骤**:
  1. 工厂对账
  2. 核对明细
  3. 生成对账单
  4. 审批与付款
- **常见问题**: 2个

### 仓储管理模块

#### 5. 面辅料出入库管理
- **难度**: 入门
- **时长**: 6分钟
- **核心步骤**:
  1. 物料采购入库
  2. 填写入库信息
  3. 物料出库发放
  4. 库存盘点
- **常见问题**: 2个
- **关键提示**:
  - 库位编号格式: 区域-货架-层-位(如A-01-02-03)
  - 按先进先出原则发放
  - 建议每月盘点一次

#### 6. 成品出库发货流程
- **难度**: 进阶
- **时长**: 8分钟
- **核心步骤**:
  1. 创建出库单
  2. 扫码拣货
  3. 包装发货
  4. 确认出库
- **常见问题**: 2个

### 小程序操作模块

#### 7. 小程序扫码工序操作
- **难度**: 入门
- **时长**: 6分钟
- **核心步骤**:
  1. 打开小程序扫码页面
  2. 扫描菲号二维码
  3. 智能工序识别
  4. 确认提交
- **常见问题**: 3个
- **关键提示**:
  - 扫码距离建议10-30cm
  - 系统自动识别工序顺序
  - 防重复保护最少30秒

### 系统设置模块

#### 8. 人员与权限管理
- **难度**: 进阶
- **时长**: 7分钟
- **核心步骤**:
  1. 创建用户账号
  2. 分配角色
  3. 配置数据权限
  4. 测试账号权限
- **常见问题**: 2个
- **关键提示**:
  - dataScope=ALL: 查看所有工厂数据
  - dataScope=FACTORY_ONLY: 仅查看本工厂数据

## 🛠️ 技术实现

### 前端实现
**文件位置**: `frontend/src/modules/system/pages/System/Tutorial/`

**核心组件**:
- `index.tsx`: 主组件(800行)
- `style.css`: 样式文件

**依赖库**:
- Ant Design 组件: Card, Collapse, Steps, Tabs, Timeline, Alert
- React Hooks: useState, useEffect
- 图标库: @ant-design/icons

**数据结构**:
```typescript
interface Tutorial {
  id: string;                    // 教程唯一ID
  title: string;                 // 教程标题
  category: string;              // 分类(production/warehouse/mobile/finance/system)
  difficulty: 'beginner' | 'intermediate' | 'advanced';  // 难度
  duration: string;              // 学习时长
  steps: TutorialStep[];         // 操作步骤数组
  videoUrl?: string;             // 视频链接(可选)
  faqs?: FAQ[];                  // 常见问题(可选)
  tags: string[];                // 标签数组
}

interface TutorialStep {
  title: string;                 // 步骤标题
  description: string;           // 步骤描述
  image?: string;                // 截图(可选)
  tips?: string[];               // 提示数组(可选)
}
```

### 路由配置
**路径**: `/system/tutorial`  
**权限码**: `MENU_TUTORIAL`  
**菜单位置**: 系统设置 → 系统教学

**配置文件**:
- `frontend/src/routeConfig.ts`: 路径和权限定义
- `frontend/src/App.tsx`: 路由注册
- `frontend/src/modules/system/index.tsx`: 组件导出

### 权限管理
**SQL脚本**: `scripts/add_tutorial_permission.sql`

**权限配置**:
1. 在字典表添加权限码 `MENU_TUTORIAL`
2. 为管理员角色分配权限
3. 其他角色在【角色管理】手动配置

## 📖 使用指南

### 管理员操作

#### 1. 开启教学功能
```sql
-- 执行SQL脚本添加权限
mysql -u root -p < scripts/add_tutorial_permission.sql
```

#### 2. 分配权限
- 进入【系统设置】→【角色管理】
- 编辑目标角色,勾选「系统教学」权限
- 保存后用户重新登录生效

#### 3. 自定义教程(未来版本)
- 后台管理界面添加/编辑教程
- 支持Markdown编辑
- 支持视频上传

### 普通用户操作

#### 1. 访问教学中心
- 登录系统后,点击左侧菜单【系统设置】→【系统教学】
- 或在首页帮助入口快速访问

#### 2. 搜索教程
- 在搜索框输入关键词(如"订单"、"扫码")
- 或点击分类按钮筛选

#### 3. 学习教程
- 点击教程卡片展开详情
- 按步骤阅读操作指引
- 查看常见问题解答
- 观看视频演示(如有)

#### 4. 反馈问题
- 点击底部「意见反馈」按钮
- 提交使用问题或改进建议

## ❓ 常见问题

### Q1: 为什么看不到系统教学菜单?
**A**: 可能是权限未配置。请联系管理员在【角色管理】中为你的角色添加「系统教学」权限。

### Q2: 可以下载教程PDF吗?
**A**: 当前版本暂不支持PDF下载,未来会添加「导出PDF」功能。可以点击底部「下载用户手册」获取完整文档。

### Q3: 视频教程无法播放?
**A**: 视频播放功能正在开发中,敬请期待。当前可先阅读文字步骤和常见问题。

### Q4: 可以提交新的教程建议吗?
**A**: 当然可以!点击「意见反馈」按钮,提交你希望看到的教程主题,我们会优先制作。

### Q5: 教程内容多久更新一次?
**A**: 系统升级时会同步更新教程。建议每月访问一次教学中心,查看是否有新教程发布。
