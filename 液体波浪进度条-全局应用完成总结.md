#### 2.4 **进度详情页（2处）** ✅

**文件**: `frontend/src/modules/production/pages/Production/ProgressDetail/index.tsx`

**修改内容**: 页面有两处使用 UniversalCardView，均已更新

```typescript
progressConfig={{
  calculate: (record: ProductionOrder) => {
    const prices = record.progressNodeUnitPrices || [];
    if (prices.length === 0) return 0;
    const completedCount = prices.filter((p) =>
      (p.completedQuantity || 0) >= (p.quantity || 0)
    ).length;
    return Math.round((completedCount / prices.length) * 100);
  },
  getStatus: (record: ProductionOrder) => {
    const shipDate = record.expectedShipDate;
    if (!shipDate) return 'normal'; // ✅ 修改
    const now = new Date();
    const delivery = new Date(shipDate);
    const diffDays = Math.ceil((delivery.getTime() - now.getTime()) / (1000 * 60 * 60 * 24));
    if (diffDays < 0) return 'danger';
    if (diffDays <= 3) return 'warning';
    return 'normal'; // ✅ 修改
  },
  show: true,
  type: 'liquid', // ✅ 新增
}}
```

**状态规则**:
- 🟢 **normal**: 无交期或交期充足
- 🟡 **warning**: 3天内临近交期
- 🔴 **danger**: 已逾期

## 🎨 视觉效果

### 进度条类型对比

| 类型 | 外观 | 使用场景 |
|------|------|----------|
| **胶囊进度条** (默认) | 椭圆形胶囊，纯色填充 | 不需要动画的简单进度 |
| **液体波浪进度条** (新) | 波浪动态流动，脉冲边框 | 需要视觉吸引力的核心数据 |

### 状态颜色系统

| 状态 | 颜色 | 含义 | 典型场景 |
|------|------|------|----------|
| **normal** | 🟢 绿色 | 正常/完成 | 充足交期、已完成任务 |
| **warning** | 🟡 黄色 | 警告/进行中 | 临近交期、部分完成 |
| **danger** | 🔴 红色 | 危险/逾期 | 已逾期、未开始 |

### 特殊行为

- **进度 >= 100%**: 波浪动画自动停止，显示完成状态
- **进度 < 100%**: 波浪持续流动，脉冲边框闪烁
- **无状态**: 默认使用 normal（绿色）

## 🔍 技术细节

### 状态类型统一

**修改前**:
```typescript
// 不同页面使用不同的状态值
'success' | 'warning' | 'danger'  // 部分页面
'default' | 'warning' | 'danger'  // 部分页面
```

**修改后**:
```typescript
// 全局统一为 LiquidProgressBar 的状态系统
'normal' | 'warning' | 'danger'  // 所有页面
```

### 类型安全保障

```typescript
// UniversalCardView/index.tsx
export interface CardProgressConfig {
  calculate: (record: any) => number;
  getStatus?: (record: any) => 'normal' | 'warning' | 'danger'; // ✅ 类型约束
  show?: boolean;
  type?: 'capsule' | 'liquid';
}
```

TypeScript 编译器会在开发时捕获类型错误，确保状态值正确。
