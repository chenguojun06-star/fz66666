### 2. fix-hardcoded-colors.sh

**功能**: 批量替换硬编码颜色为 CSS 变量

**颜色映射**（24 种）:
```javascript
// 文本颜色
'#333'    → 'var(--neutral-text)'
'#666'    → 'var(--neutral-text-secondary)'
'#999'    → 'var(--neutral-text-disabled)'
'#8c8c8c' → 'var(--neutral-text-disabled)'

// 主色
'#1890ff' → 'var(--primary-color)'
'#40a9ff' → 'var(--primary-color-light)'

// 成功色
'#52c41a' → 'var(--success-color)'
'#73d13d' → 'var(--success-color-light)'

// 错误色
'#f5222d' → 'var(--error-color)'
'#ff4d4f' → 'var(--error-color)'

// 警告色
'#faad14' → 'var(--warning-color)'
'#fa8c16' → 'var(--warning-color)'
'#d46b08' → 'var(--warning-color-dark)'

// 更多...
```

**安全机制**:
- ✅ 执行前自动创建备份分支
- ✅ 检测未提交更改并提示
- ✅ 修复后自动验证剩余违规
- ✅ 显示 Git diff 供人工审查

**回滚方法**:
```bash
git checkout color-fix-YYYYMMDD-HHMMSS
```

### 3. fix-gradients.sh

**功能**: 删除所有渐变使用，改为纯色

**处理模式**:
```tsx
// TSX 文件
background: 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)'
  → background: 'var(--primary-color)'

background: 'linear-gradient(90deg, #52c41a, #73d13d)'
  → background: 'var(--success-color)'

// CSS 文件
background: linear-gradient(...);
  → background: var(--neutral-bg);
```

**智能替换**:
- 蓝色渐变 → `var(--primary-color)`
- 绿色渐变 → `var(--success-color)`
- 橙色渐变 → `var(--warning-color)`
- 其他渐变 → `var(--neutral-bg)`

**安全机制**: 同 fix-hardcoded-colors.sh
