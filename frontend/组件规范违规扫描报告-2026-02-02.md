```tsx
// ❌ 第 XXX 行

```

**修复方案**:
```tsx
// ✅ 使用 CSS 变量
<span style={{ color: 'var(--neutral-text-secondary)', fontSize: 'var(--font-size-sm)' }}>...</span>
<span style={{ color: 'var(--neutral-text-disabled)', fontSize: 'var(--font-size-sm)' }}>无图</span>
<span style={{ color: 'var(--neutral-text-secondary)' }}>暂无封面</span>
<div><span style={{ color: 'var(--neutral-text-secondary)' }}>款号：</span></div>
// ... 其他同理
```

**映射关系**:
- `#999` → `var(--neutral-text-secondary)` (次要文字)
- `#ccc` → `var(--neutral-text-disabled)` (禁用文字)
- `#666` → `var(--neutral-text-secondary)` (次要文字)

#### 标准修复模板:

**修复前**:
```tsx
{
  title: '操作',
  width: 280,  // ❌ 宽度不统一
  render: (_: any, record: StyleInfo) => (
    <Space>  // ❌ 自定义按钮组合
      <Button size="small" onClick={() => handleEdit(record)}>编辑</Button>
      <Button size="small" danger onClick={() => handleDelete(record)}>删除</Button>
    </Space>
  )
}
```

**修复后**:
```tsx
{
  title: '操作',
  key: 'action',
  width: 150,  // ✅ 统一 150px
  fixed: 'right' as const,  // ✅ 固定在右侧
  render: (_: any, record: StyleInfo) => (
    <RowActions  // ✅ 使用 RowActions
      actions={[
        {
          key: 'edit',
          label: '编辑',
          primary: true,  // ✅ 主要操作
          onClick: () => handleEdit(record),
        },
        {
          key: 'delete',
          label: '删除',
          danger: true,  // ✅ 危险操作
          onClick: () => handleDelete(record),
        },
      ]}
    />
  )
}
```
