## 🔧 需要修复的问题

### 优先级P0：核心功能修复

#### 1. 修复 isDeprecatedEndpoint() 函数

**当前代码**（有问题）：
```typescript
export function isDeprecatedEndpoint(url: string, method?: string): boolean {
  const config: AxiosRequestConfig = { url, method: method?.toLowerCase() };
  const adapted = adaptLegacyApi(config);
  return adapted !== config; // ❌ 对象引用永远不相等
}
```

**应该改为**：
```typescript
export function isDeprecatedEndpoint(url: string, method?: string): boolean {
  const config: AxiosRequestConfig = { url, method: method?.toLowerCase() };
  const adapted = adaptLegacyApi(config);
  return adapted.url !== config.url; // ✅ 比较URL字符串
}
```

#### 3. 修复 /save 端点尾部斜杠

**当前代码**：
```typescript
'POST /api/production/order/save': (config) => {
  const hasId = config.data?.id;
  return {
    ...config,
    method: hasId ? 'put' : 'post',
    url: config.url?.replace(/\/save$/, hasId ? '' : '') // ❌ 缺少尾部斜杠
  };
}
```

**应该改为**：
```typescript
'POST /api/production/order/save': (config) => {
  const hasId = config.data?.id;
  return {
    ...config,
    method: hasId ? 'put' : 'post',
    url: config.url?.replace(/\/save$/, hasId ? '/' : '/') // ✅ 统一添加斜杠
  };
}
```
