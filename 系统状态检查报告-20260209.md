### 修复2: 自动补充 orderNo 字段（line 719-726）
```java
// 构建 orderId -> orderNo 映射
Map<String, String> orderIdToOrderNoMap = validOrders.stream()
    .collect(Collectors.toMap(
        ProductionOrder::getId,
        ProductionOrder::getOrderNo,
        (v1, v2) -> v1
    ));
```
**效果**: ✅ 如果采购任务有 orderId 但无 orderNo，自动从订单表获取

### 修复3: 允许独立采购任务（line 731-741）
```java
.filter(purchase -> {
    String orderId = purchase.getOrderId();
    // 如果没有关联订单，保留（独立采购）
    if (!StringUtils.hasText(orderId)) {
        return true;
    }
    // 如果有关联订单，检查订单是否有效
    return validOrderIds.contains(orderId);
})
```
**效果**: ✅ 允许返回 order_id 为 NULL 的独立采购任务

### 修复4: 补充缺失的 orderNo（line 742-751）
```java
.peek(purchase -> {
    // 如果 orderNo 为空，从映射表中补充
    if (!StringUtils.hasText(purchase.getOrderNo())) {
        String orderId = purchase.getOrderId();
        if (StringUtils.hasText(orderId)) {
            String orderNo = orderIdToOrderNoMap.get(orderId);
            if (orderNo != null) {
                purchase.setOrderNo(orderNo);
            }
        }
    }
})
```
**效果**: ✅ 动态补充 orderNo 字段
