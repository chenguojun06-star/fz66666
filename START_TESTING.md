## 🐛 遇到问题？

### 常见问题

**Q1: 找不到测试按钮？**
- 确认首页快捷入口是否显示
- 检查 app.json 是否注册了 scan-test 页面
- 重新编译小程序

**Q2: 扫码没反应？**
- 检查工厂是否选择
- 检查用户是否登录
- 查看控制台错误日志

**Q3: 模拟扫码失败？**
- 确认 debugMode 已开启
- 检查 ScanHandler 是否正确导入
- 查看 Network 请求是否成功

### 获取帮助

1. 查看测试日志：
   ```
   微信开发者工具 → Console → 搜索 "[ScanTest]"
   ```

2. 查看网络请求：
   ```
   微信开发者工具 → Network → 筛选 "production"
   ```

3. 提交问题：
   - 在 TEST_CHECKLIST.md 记录问题
   - 附上截图和日志
   - 标注严重程度
