## 1. 组件库概述

### 1.1 目标

为智能化功能提供一套统一、可复用的前端组件库，包括：

- **智能卡片组件**：展示智能建议、风险预警、健康度评分
- **AI 对话组件**：小云智能助手的对话界面
- **数据可视化组件**：图表、进度条、状态指示器
- **操作组件**：一键执行、批量操作、审批流程

### 1.2 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 18.x | UI 框架 |
| TypeScript | 5.x | 类型支持 |
| Ant Design | 6.x | 基础组件库 |
| Tailwind CSS | 3.x | 样式工具 |
| Recharts | 2.x | 图表库 |
| React Markdown | 9.x | Markdown 渲染 |
| AI SDK | 3.x | AI 对话集成 |

## 2. 设计原则

### 2.1 一致性原则

```
┌─────────────────────────────────────────────────────────────────┐
│                    设计一致性规范                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   颜色一致性                                                     │
│   ├── 风险级别：LOW(绿) / MEDIUM(橙) / HIGH(深橙) / CRITICAL(红)│
│   ├── 健康度：优秀(绿) / 良好(浅绿) / 警告(橙) / 危险(红)       │
│   └── 操作按钮：主要(蓝) / 危险(红) / 默认(灰)                  │
│                                                                 │
│   布局一致性                                                     │
│   ├── 卡片圆角：8px                                             │
│   ├── 内边距：16px                                              │
│   ├── 间距：12px                                                │
│   └── 阴影：0 2px 8px rgba(0,0,0,0.08)                          │
│                                                                 │
│   交互一致性                                                     │
│   ├── 悬停效果：阴影加深 + 轻微上移                             │
│   ├── 点击效果：涟漪动画                                        │
│   ├── 加载状态：骨架屏 + 加载动画                               │
│   └── 错误状态：红色边框 + 错误提示                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 可访问性原则

- **键盘导航**：所有交互元素支持 Tab 键导航
- **屏幕阅读器**：提供 aria-label 属性
- **颜色对比度**：文本与背景对比度 > 4.5:1
- **焦点指示器**：清晰的焦点样式

### 2.3 性能原则

- **懒加载**：非首屏组件延迟加载
- **虚拟列表**：长列表使用虚拟滚动
- **防抖节流**：输入和滚动事件优化
- **缓存策略**：合理使用 React Query 缓存

## 3. 核心组件

### 3.1 IntelligenceCard 智能卡片

#### 功能描述

展示智能建议、风险预警、健康度评分等信息的卡片组件。

#### Props 定义

```typescript
interface IntelligenceCardProps {
  insight: IntelligenceInsight;
  onAction?: (action: NextAction) => void;
  onDetail?: () => void;
  compact?: boolean;
  showTrace?: boolean;
  className?: string;
}

interface IntelligenceInsight {
  id: string;
  type: 'ORDER_RISK' | 'STOCK_ALERT' | 'FINANCE_DIFF' | 'QUALITY_ISSUE' | 'SYSTEM_ALERT';
  title: string;
  summary: string;
  riskLevel: 'LOW' | 'MEDIUM' | 'HIGH' | 'CRITICAL';
  reasons: string[];
  nextActions: NextAction[];
  healthScore?: number;
  confidence?: number;
  traceId?: string;
  createdAt?: string;
  metadata?: Record<string, any>;
}

interface NextAction {
  id: string;
  type: 'REMIND' | 'DISPATCH' | 'CREATE_PURCHASE' | 'APPROVE' | 'NOTIFY' | 'VIEW_DETAIL';
  label: string;
  primary?: boolean;
  danger?: boolean;
  payload?: Record<string, any>;
  confirmMessage?: string;
}
```

#### 使用示例

```tsx
import { IntelligenceCard } from '@/components/intelligence';

function OrderDetailPage() {
  const { data: insight } = useOrderInsight(orderId);
  
  const handleAction = async (action: NextAction) => {
    switch (action.type) {
      case 'REMIND':
        await remindOrder(orderId);
        message.success('催办已发送');
        break;
      case 'VIEW_DETAIL':
        navigate(`/orders/${orderId}/detail`);
        break;
    }
  };
  
  return (
    <div>
      <h2>订单详情</h2>
      
      {insight && (
        <IntelligenceCard
          insight={insight}
          onAction={handleAction}
          onDetail={() => navigate(`/intelligence/insights/${insight.id}`)}
        />
      )}
    </div>
  );
}
```

### 3.3 IntelligencePanel 智能面板

#### 功能描述

智能驾驶舱的核心面板组件，展示预警列表、任务列表、统计数据。

#### Props 定义

```typescript
interface IntelligencePanelProps {
  defaultTab?: 'alerts' | 'tasks' | 'insights';
  showStatistics?: boolean;
  refreshInterval?: number;
  onAlertClick?: (alert: Alert) => void;
  onTaskClick?: (task: Task) => void;
}

interface Alert {
  id: string;
  type: string;
  level: 'LOW' | 'MEDIUM' | 'HIGH' | 'CRITICAL';
  title: string;
  message: string;
  source: string;
  createdAt: string;
  status: 'PENDING' | 'PROCESSING' | 'RESOLVED' | 'IGNORED';
}

interface Task {
  id: string;
  type: string;
  title: string;
  description: string;
  assignee: string;
  dueTime: string;
  status: 'PENDING' | 'IN_PROGRESS' | 'COMPLETED' | 'CANCELLED';
  priority: 'LOW' | 'MEDIUM' | 'HIGH';
}
```

#### 使用示例

```tsx
import { IntelligencePanel } from '@/components/intelligence';

function IntelligenceCenterPage() {
  const handleAlertClick = (alert: Alert) => {
    navigate(`/alerts/${alert.id}`);
  };
  
  const handleTaskClick = (task: Task) => {
    navigate(`/tasks/${task.id}`);
  };
  
  return (
    <div className="intelligence-center">
      <h1>智能驾驶舱</h1>
      
      <IntelligencePanel
        defaultTab="alerts"
        showStatistics
        refreshInterval={30000}
        onAlertClick={handleAlertClick}
        onTaskClick={handleTaskClick}
      />
    </div>
  );
}
```
