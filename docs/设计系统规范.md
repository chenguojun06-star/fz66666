# 服装供应链管理系统 - 设计系统规范 v1.0

> **创建日期**：2026-01-28  
> **维护者**：开发团队  
> **原则**：统一、简洁、舒适、高性能

---

## 📐 核心设计原则

### 1. 高度统一性
- ✅ 所有按钮使用**胶囊形**（border-radius: 999px）
- ✅ 所有卡片使用**玻璃拟态**效果
- ✅ 所有动画使用**统一缓动函数**
- ✅ 所有颜色遵循**预定义变量**

### 2. 舒适交互
- ⏱️ 动画时长：200-400ms（快速不拖沓）
- 🎯 Hover反馈：上浮 + 阴影增强
- 🎨 颜色舒适：自然绿色系为主
- 💫 缓动柔和：cubic-bezier 曲线

### 3. 性能优先
- ⚡ 纯CSS实现，无JS依赖
- 🚀 GPU加速（transform、opacity）
- 📦 按需加载，避免冗余

## 🔘 按钮规范（镂空胶囊形统一）

### 主按钮（Ant Design Primary）
```tsx
<Button type="primary">查询</Button>
<Button type="primary" icon={<SearchOutlined />}>搜索</Button>
```

**浅色主题**：
- ⚪ 背景：白色（`#ffffff`）
- 🔵 边框：2px 浅蓝色（`#40a9ff`）
- 🎨 文字：浅蓝色（`#40a9ff`）
- ✨ Hover：淡蓝背景（`#f0f9ff`）+ 深蓝边框（`#1890ff`）
- 💫 阴影：`0 2px 8px rgba(64, 169, 255, 0.15)`
- 📐 圆角：999px（胶囊形）

**深色主题**：
- 🌑 背景：深灰半透明（`rgba(26, 32, 39, 0.8)`）
- 🔵 边框：2px 亮蓝色（`#69b1ff`）
- 🎨 文字：亮蓝色（`#69b1ff`）
- ✨ Hover：蓝色半透明背景（`rgba(64, 169, 255, 0.1)`）+ 最亮蓝边框（`#91d5ff`）

### 次要按钮（Default）
```tsx
<Button>取消</Button>
<Button>重置</Button>
```
- ⚪ 背景：白色
- ⚫ 边框：灰色（`#e5e7eb`）
- 🎨 Hover：淡蓝背景 + 蓝色边框

### 幽灵按钮（Ghost）
```tsx
<Button ghost>查看详情</Button>
```
- 透明背景
- 蓝色边框 + 文字
- Hover：淡蓝背景

### 尺寸变体
```tsx
<Button size="small">小按钮</Button>
<Button size="middle">默认</Button>  {/* 默认尺寸 */}
<Button size="large">大按钮</Button>
```

## 🌊 动画规范

### 时长
```css
--duration-fast: 0.2s   /* 快速交互 */
--duration-base: 0.3s   /* 标准动画 */
--duration-slow: 0.4s   /* 慢速过渡 */
```

### 缓动函数
```css
--ease-smooth: cubic-bezier(0.16, 1, 0.3, 1)      /* 平滑 */
--ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1)  /* 弹性 */
--ease-out: cubic-bezier(0.33, 1, 0.68, 1)        /* 缓出 */
```

### 页面进入动画
```css
animation: slideInUp 0.6s var(--ease-smooth);
```

### 悬停动画
```css
transition: all 0.3s var(--ease-smooth);
```

## ✅ 使用检查清单

### 新增页面必做
- [ ] 引入 `design-system.css`
- [ ] 使用 `page-container-glass` 容器
- [ ] 所有按钮使用 `btn-pill` 类
- [ ] 所有卡片使用 `card-glass` 类
- [ ] 颜色使用 CSS 变量
- [ ] 动画使用预定义缓动

### 代码审查检查
- [ ] 无硬编码颜色（如 `#22c55e` 应改为 `var(--color-primary)`）
- [ ] 无自定义圆角（统一使用 `var(--radius-*)`）
- [ ] 无自定义缓动（统一使用 `var(--ease-*)`）
- [ ] 按钮必须是胶囊形
- [ ] 卡片必须有玻璃拟态效果

## 🚫 禁止事项

❌ **禁止使用的样式**：
- ❌ 实心填充按钮（主按钮必须镂空白底+彩色边框）
- ❌ 深色渐变按钮（仅进度条可用深色渐变）
- ❌ 直角按钮（border-radius < 12px）
- ❌ 纯色卡片背景（必须半透明 + 模糊）
- ❌ 硬编码颜色值
- ❌ 自定义动画时长（必须用变量）
- ❌ 过度复杂的动画（影响性能）

❌ **禁止的操作**：
- 随意修改设计系统变量
- 在单个页面创建独立样式系统
- 不经讨论更改全局规范

## 🌓 深色主题规范

### 颜色适配原则
1. **按钮颜色**：浅色主题用浅蓝（`#40a9ff`），深色主题用亮蓝（`#69b1ff`）
2. **背景透明度**：深色主题按钮背景使用半透明深灰（`rgba(26, 32, 39, 0.8)`）
3. **文字对比度**：确保文字与背景对比度 ≥ 4.5:1
4. **边框可见性**：深色主题边框颜色需比浅色主题更亮

### 自动切换
```css
/* 浅色主题 */
.ant-btn-primary {
  background: #ffffff;
  border-color: #40a9ff;
  color: #40a9ff;
}

/* 深色主题 */
[data-theme="dark"] .ant-btn-primary {
  background: rgba(26, 32, 39, 0.8);
  border-color: #69b1ff;
  color: #69b1ff;
}
```
