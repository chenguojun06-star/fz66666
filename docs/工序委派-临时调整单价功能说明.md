# 工序委派功能 - 临时调整单价

**完成时间**: 2026-01-31  
**功能模块**: 生产订单 - 工序详情

## 功能概述

在工序详情弹窗的"工序委派"Tab中，添加了**临时调整单价**的功能，允许用户为委派给不同工厂的工序设置自定义单价。

## 功能特性

### 前端实现

#### 1. UI组件（frontend/src/modules/production/pages/Production/List/index.tsx）

**新增字段**：
- 在委派表格中添加了"委派单价"列
- 使用 `InputNumber` 组件，支持小数输入（精度2位）
- 显示格式：¥ 前缀，最小值0，步进0.01

**状态管理**：
```typescript
const [delegationData, setDelegationData] = useState<Record<string, { 
  factoryId?: string; 
  unitPrice?: number 
}>>({});
```

**表格列**（7列）：
1. 工序节点
2. 完成状态
3. 委派工厂（Select下拉）
4. **委派单价**（InputNumber输入框） ← 新增
5. 操作人
6. 完成时间
7. 操作按钮

#### 2. 保存功能

**API调用**：
```typescript
const saveDelegation = async (nodeKey: string, orderId: string) => {
  const data = delegationData[nodeKey];
  if (!data?.factoryId) {
    message.warning('请选择委派工厂');
    return;
  }

  await api.post('/production/order/delegate-process', {
    orderId,
    processNode: nodeKey,
    factoryId: data.factoryId,
    unitPrice: data.unitPrice || 0
  });
  
  message.success('工序委派保存成功');
};
```

**触发方式**：
- 点击每行的"保存"按钮
- 验证：必须选择工厂，单价可选（默认0）
- 保存后自动刷新工序状态

## 使用流程

1. **打开工序详情**
   - 在生产订单列表，点击订单的"工序详情"按钮
   - 弹出ResizableModal（60vw × 60vh）

2. **切换到委派Tab**
   - 点击"工序委派"标签页
   - 显示4个工序节点：裁剪、车缝、尾部、入库

3. **设置委派信息**
   - 选择委派工厂（必填）
   - 输入委派单价（可选，默认0）
   - 可以为每个工序设置不同的工厂和单价

4. **保存委派**
   - 点击对应工序行的"保存"按钮
   - 显示成功提示
   - 自动刷新工序状态

5. **查看历史**
   - 委派记录存储在订单的 `nodeOperations` 字段
   - 包含完整的操作历史和审计信息

## 技术细节

### 前端组件

**InputNumber配置**：
```tsx
<InputNumber
  placeholder="单价"
  size="small"
  min={0}
  step={0.01}
  precision={2}
  prefix="¥"
  style={{ width: '100px' }}
  value={delegationData[node.key]?.unitPrice}
  onChange={(value) => {
    setDelegationData(prev => ({
      ...prev,
      [node.key]: { ...prev[node.key], unitPrice: value || undefined }
    }));
  }}
/>
```

**特性**：
- 最小值：0
- 步进：0.01元
- 精度：2位小数
- 前缀：¥ 符号
- 宽度：100px
- 受控组件：value + onChange

### 后端处理

**事务管理**：
- 使用 `@Transactional` 确保数据一致性
- 委派失败时自动回滚

**日志记录**：
```java
log.info("工序委派成功 - 订单:{}, 工序:{}, 工厂:{}, 单价:{}", 
    orderId, processNode, factoryId, unitPrice);
```

**错误处理**：
- 订单不存在 → 抛出RuntimeException
- 工厂不存在 → 前端验证（从工厂列表选择）
- 单价为空 → 默认为0
