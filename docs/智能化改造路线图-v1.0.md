#### 健康度评分公式

```
总分 = 进度分 × 40% + 交付分 × 35% + 采购分 × 25%

进度分 = 100 - (预期进度 - 实际进度) / 预期进度 × 100
交付分 = 100 - |实际进度 - 时间预期进度|
采购分 = 100 - (缺料数 × 20 + 延迟到货数 × 10)
```

## 8. 风险管理

|------|------|------|---------|---------|
| 数据库迁移失败 | 高 | 中 | 完整备份 + 增量同步 | 回滚到 MySQL |
| 向量检索性能差 | 中 | 中 | 预热索引 + 分片策略 | 降级到 Qdrant |
| AI 响应延迟 | 中 | 高 | 超时降级 + 流式响应 | 缓存常见问题 |
| Spring Boot 升级失败 | 高 | 低 | 分模块升级 + 充分测试 | 回滚到 2.7 |

| 用户不采纳 | 中 | 中 | 可解释性增强 | 人工审核 |
