# 前端 UI 全面优化 — 设计规范文档

> 日期：2026-05-20 | 方向：精致商务风（Notion/Linear 路线）
> 仪表盘（Dashboard）不改动

## 核心原则

- **不改功能逻辑**，不改操作方式，不改组件 API
- **只改视觉层**：CSS + 间距 + 布局分区
- **每步编译验证**：`npx tsc --noEmit` → 0 errors
- **进度球保持不动**：LiquidProgressLottie/LiquidProgressBar 不修改

---

## 一、设计系统更新

| 规范项 | 原值 | 改为 | 理由 |
|--------|------|------|------|
| 页面背景 | 渐变 + 毛玻璃碎片 | `#f8fafc` 纯色 | 减少视觉噪音 |
| 卡片背景 | `rgba(255,255,255,0.8)` + `blur(12px)` | `#fff` + `1px #e5e7eb` 边框 | 扁平化 |
| 卡片圆角 | 12-16px | 6px | 行业 4-8px 标准 |
| 卡片阴影 | 多层阴影 | `0 1px 3px rgba(0,0,0,.06)` | 去臃肿感 |
| 间距基准 | 8pt 变量存在但未强制执行 | 强 4/8/12/16/20/24/32/40 | 行业标准 |
| 表格行高 | 38-52px 不统一 | 44px | 行业标准 44-48px |
| 输入框高度 | 32px | 36px | 可用性提升 |
| 表头高度 | 不一致 | 36px (#fafafa, #9ca3af) | 统一 |
| 斑马纹 | 较深 | #fafafa / #fff | 极淡 |
| 分隔线 | 不一致 | 1px #f0f0f0 | 仅横线 |
| hover 行 | 不一致 | #f3f4f6 | 淡灰 |

---

## 二、反臃肿 6 规则

1. **三区分隔** — 每页 4 Zone：标题→统计→筛选→表格，区间 16px 白条
2. **一行一重点** — 表格行最多 1 彩色标签 + 1 进度指示器
3. **卡片 ≤5 字段** — 统计卡片：标签(10px) + 数值(22px)，就 2 行
4. **筛选区折叠** — 条件 > 3 个时只展前 3 +「更多筛选」
5. **空状态统一** — EmptyState 组件（灰图标 + 一行说明）
6. **删多余** — 装饰分割线、多余图标、重复标题、冗余边框直接删

---

## 三、页面模板

```
Zone 1 — 标题区：标题 + 副标题 + 操作按钮
  ↓ 16px
Zone 2 — 统计区（可选）：≤4 卡片，每卡 2 行
  ↓ 16px
Zone 3 — 筛选区：搜索 + ≤3 筛选 +「更多筛选」
  ↓ 16px
Zone 4 — 表格区：ResizableTable，行高 44px
```
