# 多语言共享词条说明（PC + 小程序）

## 目标
- PC 端与小程序共用同一份翻译源，避免两端术语不一致。

## 单一词条源
- 源目录：`shared-locales/source/`
- 文件：
  - `zh-CN.json`
  - `en-US.json`
  - `vi-VN.json`
  - `km-KH.json`

## 同步命令
```bash
node scripts/sync-locales.js
```

## 生成产物
- PC：`frontend/src/i18n/locales.generated.ts`
- 小程序：`miniprogram/utils/i18n/locales.generated.js`

## 小程序适配层
- 文件：`miniprogram/utils/i18n/index.js`
- 能力：
  - `getLanguage()`
  - `setLanguage(lang)`
  - `t(keyPath, lang?)`

## 开发约定
1. 新增文案时，先改 `shared-locales/source/*.json`。
2. 执行 `node scripts/sync-locales.js`。
3. 两端只引用生成产物，不手改 generated 文件。
4. 业务术语需统一（如“工序/尾部/对账/样衣”）。
