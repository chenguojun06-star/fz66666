## 🎨 深色主题设计原则

### 1. 背景色层次
```css
/* 3层背景深度 */
页面背景:     rgba(26, 29, 36, 0.6)  /* 最深 */
卡片背景:     rgba(26, 29, 36, 0.4)  /* 次深 */
表头/按钮:    rgba(255, 255, 255, 0.03-0.05)  /* 最浅 */
```

### 2. 边框和阴影
```css
边框: rgba(255, 255, 255, 0.1)      /* 半透明白色 */
阴影: 0 2px 8px rgba(0, 0, 0, 0.4)  /* 深色阴影 */
```

### 3. 文字颜色
```css
主文字:   var(--neutral-text)        /* #f5f5f5 */
次文字:   var(--neutral-text-light)  /* #d1d5db */
辅助文字: var(--neutral-text-lighter) /* #9ca3af */
```

### 4. 交互效果
- Hover: 提升背景亮度（增加白色透明度）
- Active: 边框颜色变亮
- 焦点: 蓝色高亮（保持品牌色）

## 📝 技术亮点

1. **完全使用CSS变量** - 所有颜色都通过 `:root[data-theme="dark"]` 覆盖，支持主题热切换
2. **保持视觉层次** - 通过不同透明度的深灰色创建3层深度
3. **语义化配色** - 8种统计卡片类型各有独特的深色主题配色（蓝/青/绿/红/黄/紫/橙）
4. **无硬编码** - 所有文字颜色使用 `var(--neutral-text-*)` 变量
5. **平滑过渡** - 所有交互都有 `transition` 动画效果
