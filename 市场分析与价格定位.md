## 📊 第一部分：国内竞品价格对标

### 鼎普（业界标杆）

**企业卡片**：
- 上市公司，2001年成立，老牌服装MES
- 产品线：鼎普MES、APS（高级排期）、WMS（仓管）
- 定价方式：SaaS按用户数 + 功能模块订阅
- 目标客户：大中型服装工厂、上市企业

**价格体系**：

| 规模 | 用户数 | 基础版 | 标准版 | 高级版 | 说明 |
|------|--------|--------|--------|--------|------|
| 小工厂 | 10-50人 | 8000/月 | 12000/月 | 16000/月 | 5步计算 |
| 中工厂 | 50-200人 | 15000/月 | 22000/月 | 32000/月 | 10步计算 |
| 大工厂 | 200-500人 | 25000/月 | 38000/月 | 52000/月 | 20步计算 |
| 超大 | 500+人 | **议价** | **议价** | **议价** | 百万级合同 |

**按功能模块加费**：
- APS高级排期：+5000/月
- WMS仓管：+3000-5000/月
- 品质管理：+2000/月
- CRM客服：+3000/月
- 财税接口：+4000/月

**年度费用范例**：
- 50人工厂：`12000×12 = 14.4万/年`
- 200人工厂：`22000×12 = 26.4万/年`
- 500人工厂：30万+/年

**实施周期**：3-6个月（需二开调整）
**实施成本**：3-10万（咨询、培训、数据迁移）
**总成本**：50人工厂首年 **18万-22万**

## 💰 第二部分：价格定位建议

### 定价模式选择

#### 方案 A：**按工厂规模 + 用户数**（推荐）

```
基础版（≤50人）：1500元/月（≈ 18000元/年）
标准版（50-200人）：3000元/月（≈ 36000元/年）
专业版（200-500人）：5000元/月（≈ 60000元/年）
企业版（500人+）：8000元/月+（议价）

按功能模块加费：
- 财税对接（开发中）：+1000元/月 或 +5000元（一次性）
- CRM模块（开发中）：+1500元/月 或 +8000元（一次性）

年度打折：年付9折，续费8折
```

**优点**：易售、易理解、竞争力强
**缺点**：难以覆盖大客户

---

#### 方案 B：**按扫码次数**（补充方案）

```
1. 免费额度：5万次/月（吸引小工厂）
2. 超额计费：0.05元/次（超出部分）
3. 月均费用：
   - 10万次 = 2500元
   - 50万次 = 22500元
   - 100万次 = 40000元

年度打折：20% off
```

**优点**：用户付费与使用量挂钩，公平
**缺点**：大用户可能流失

---

#### 方案 C：**混合定价**（最优），推荐

```
基础套餐（按工厂）：
  小工厂（≤50人）：2000元/月 ← 对标鼎普的 1/6，太便宜
  中工厂（50-200人）：4000元/月 ← 对标鼎普的 1/5.5，有竞争力
  大工厂（200-500人）：6000元/月 ← 对标鼎普的 1/6.3

实际费用计算：
  中工厂年费：4000×12 = 4.8万/年 ← vs 鼎普 26.4万（↓ 79.5%！）
  
额外增值：
  年度合同折扣：9折（5000+元/月）、8折（10000+元/月）
  功能定制费：按工时计费（500元/小时）
  部署服务费：3000元（一次性，包含数据迁移+培训）
```

**竞争力演算**：
```
用户对比预期：

鼎普50人：14.4万/年
你的系统50人：2.4万/年  ← ✅ 对标优势：↓ 83.3%

鼎普200人：26.4万/年
你的系统200人：4.8万/年  ← ✅ 对标优势：↓ 81.8%

鼎普500人：60.0万/年
你的系统500人：7.2万/年  ← ✅ 对标优势：↓ 88%
```
