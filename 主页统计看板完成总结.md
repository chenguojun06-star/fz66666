**功能**：

```typescript
// 卡片配置
样衣开发: 紫色渐变 #a78bfa → #8b5cf6 (TagsOutlined)
大货下单: 蓝色渐变 #60a5fa → #3b82f6 (ShoppingCartOutlined)
裁剪数量: 橙色渐变 #fbbf24 → #f59e0b (ScissorOutlined)
出入库量: 绿色渐变 #34d399 → #10b981 (InboxOutlined)
```

### 2. 后端API实现

**接口**：`GET /api/dashboard/top-stats?range={day|week|month|year}`

**新增文件**：
- `TopStatsResponse.java` - 响应DTO

**修改文件**：
- `DashboardController.java` - 新增端点
- `DashboardOrchestrator.java` - 业务逻辑
- `DashboardQueryService.java` - 查询接口
- `DashboardQueryServiceImpl.java` - 查询实现

**时间计算逻辑**：
```java
day   → 今日00:00 至 当前时间
week  → 本周一00:00 至 当前时间
month → 本月1日00:00 至 当前时间
year  → 今年1月1日00:00 至 当前时间
```

**数据来源**：
| 指标 | 数据表 | 字段/条件 |
|------|--------|-----------|
| 样衣开发 | t_style_info | is_sample=1, create_time范围内 |
| 大货下单 | t_production_order | create_time范围内的订单数 |
| 裁剪数量 | t_production_order | cutting_quantity求和 |
| 出入库量 | t_product_warehousing | qualified_quantity + unqualified_quantity |

## 🎨 设计特点

### 视觉风格
- **形状**：胶囊形（border-radius: 24px）
- **配色**：柔和渐变色，4种不同色系
- **排版**：图标+数字+标签，左右布局
- **间距**：16px卡片间距，20px内边距

### 交互效果
- **悬浮**：上浮4px + 阴影加深
- **选中**：时间按钮白色背景+蓝色文字+阴影
- **过渡**：all 0.3s ease

### 响应式设计
- **桌面**（>1200px）：4列网格
- **平板**（768-1200px）：2列网格
- **移动**（<768px）：1列堆叠

### 深色主题
- 时间选择器：#f3f4f6 → #374151
- 激活按钮：white → #4b5563
- 文字颜色：#6b7280 → #9ca3af
- 阴影强度：0.1 → 0.3
