# 主页数据看板 - 快速测试指南

## 🚀 快速启动

### 1. 启动后端（可选）
```bash
cd backend
source ../.run/backend.env
mvn spring-boot:run
```
> 注意：当前使用虚拟数据，后端可以不启动

### 2. 启动前端（必须）
```bash
cd frontend
npm run dev
```
访问：http://localhost:5173

## 👀 查看效果

打开浏览器后，主页从上到下依次显示：

### ✅ 第1部分：搜索框
盲收搜索框

## ❓ 常见问题

**Q: 折线图没有数据？**
A: 检查浏览器控制台是否有报错，确认虚拟数据代码是否正常执行。

**Q: 表格没有分页？**
A: 虚拟数据共25条，前10条在第1页，点击页码查看后续数据。

**Q: 质检看板位置不对？**
A: 应该在4个统计卡片正下方，数据分析区正上方。

**Q: 想看真实数据？**
A: 需要启动后端服务，并按上述步骤切换API调用。
